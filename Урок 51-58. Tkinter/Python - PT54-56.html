Практическое Задание 54.1

Создайте класс "Сотрудник". Этот класс должен хранить поля для идентификатора, имени, фамилии и списка отзывов.
Для класса определите строковое представление, конструктор и свойства.
Для класса определите CRUD для отзыва.

Создайте класс "База данных сотрудников". Этот класс должен хранить поле для списка сотрудников.
Для класса определите свойство.
Реализуйте CRUD для сотрудников.
Реализуйте возможность вызывать CRUD у сотрудника для отзыва.
