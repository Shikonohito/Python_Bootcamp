Домашнее Задание 56.1

На основе Вашего финального проекта создайте визуальное приложение. Приложение должно использовать Label, Entry, Button, Listbox, а также Ваши классы.
Реализуйте CRUD для своей базы данных. Все изменения должны отображаться в Listbox. Приложение должно содержать как минимум два фрейма. Переключение фреймов должно быть через меню.

В этом домашнем задании будет проверяться связка фронтенда и бэкенда.
