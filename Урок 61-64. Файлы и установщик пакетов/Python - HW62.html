Домашнее Задание 62.1

На основе Вашего финального проекта реализуйте сохранение и загрузку базы данных в файл внутри Вашего класса. Также реализуйте в графическом приложении возможность сохранять и загружать базу данных.

На это домашнее задание Вы отправляете свой итоговый вариант финального проекта.

Как отправлять:

  1. Найдите папку с проектом.
  2. Заархивируйте папку с проектом.
  3. Отправьте полученный архив.

Как найти место хранения проекта:

  1. Откройте проект в PyCharm.
  2. Слева нажмите Правой Кнопкой Мыши по папке с названием проекта.
  3. Выберите "Open In".
  4. Выберите "Explorer".

Как заархивировать:

  1. Нажмите Правой Кнопкой Мыши по папке с проектом.
  2. Наведите курсор на "Compress to...".
  3. Выберите "ZIP File".
