Практическое Задание 9.10

Teston Lebra хочет сделать вклад в банк под годовой процент, но не хочет долго высчитывать, сколько денег у него будет через несколько лет. Напишите программу, которая запрашивает с консоли начальную сумму вклада, годовую процентную ставку и количество лет, затем выводит в консоль сумму вклада через указанное количество лет.

Программа должна считать по формуле:
<сумма_вклада> = <начальная_сумма_вклада> * ((1 + <годовая_процентная_ставка> / 100) ** <количество_лет>)

Консоль:

Введите начальную сумму вклада: 1000
Введите годовую процентную ставку: 5
Введите количество лет: 5

Сумма вклада через 5 лет: 1276.28
