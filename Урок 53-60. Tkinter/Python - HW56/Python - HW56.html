Домашнее Задание 56.1

Создайте класс "Таксист". Этот класс должен хранить поля для идентификатора, имени, фамилии и списка оценок.
Для класса определите строковое представление, конструктор и свойства.
Для класса определите метод добавления оценки в список оценок.
Для класса определите метод получения средней арифметической оценки.

Создайте класс "База данных таксистов". Этот класс должен хранить поле для списка таксистов.
Для класса определите свойство.
Реализуйте CRUD.
Для класса определите метод добавления оценки по идентификатору таксиста.
Для класса определите метод получения средней арифметической оценки по идентификатору таксиста.

Домашнее Задание 56.2

Создайте визуальное приложение для работы с классами из Домашнего Задания 56.1. В приложении должна быть возможность добавить нового таксиста, показать, изменить и удалить таксиста. Также должна быть возможность выбрать таксиста в списке и отдельно добавить ему оценку. Визуальное приложение должно содержать Label, Entry, Button, Listbox.
