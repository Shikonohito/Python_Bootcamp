Домашнее Задание 10.1

Напишите программу, которая запрашивает с консоли среднюю скорость автомобиля в км/ч. Затем программа выводит в консоль меню:

2. Определить расстояние
После этого программа запрашивает с консоли выбор пункта.

Выбран пункт 1:

  • Программа запрашивает с консоли расстояние в км.

2. Определить расстояние

================================
Введите расстояние (км): 200

Домашнее Задание 10.10

Teston Lebra попросили написать прототип программы, которая запрашивает с консоли 16-значный номер карты, месяц, год и CVV/CVC, затем программа сверяет данные с тестовыми и выводит в консоль "Операция проведена успешно." или "Ошибка ввода.". Он написал программу, но она работает неверно. Исправьте программу так, чтобы она работала верно только с тестовыми данными. Остальные данные должны приводить к ошибке ввода, и рядом напишите комментарии о том, почему изменённый вариант должен работать верно.

Код:

card_id = int(input("Введите 16-значный код карты: "))
month = int(input("Введите месяц: "))
year = int(input("Введите год: "))
cvv_cvc = int(input("Введите CVV/CVC: "))

test_card_id = "5869586958695869"
test_month = "03"
test_year = "28"
test_cvv_cvc = "094"

check_success = 0

if test_card_id == str(card_id):
    check_success += 1
if test_month == str(month):
    check_success += 1
if test_year == str(year):
    check_success += 1
if test_cvv_cvc == str(cvv_cvc):
    check_success += 1

print()
if check_success == 4:
    print("Операция проведена успешно.")
if check_success != 4:
    print("Ошибка ввода.")

Пример работы в консоли:

Введите 16-значный код карты: 5869586958695869
Введите месяц: 03
Введите год: 28
Введите CVV/CVC: 094
