Напишите программу, которая запрашивает с консоли уровень загрязнения воздуха (в микрограммах на кубический метр), уровень шума (в децибелах), концентрацию углекислого газа (в частях на миллион), а также уровень радиации (в микрозивертах в час), затем выводит в консоль оценку качества окружающей среды.

Градация оценки:

  • Отличное: все параметры находятся в пределах нормы.
  • Хорошее: 1 параметр выходит за пределы нормы.
  • Удовлетворительное: 2 параметра выходят за пределы нормы.
  • Плохое: 3 параметра выходят за пределы нормы.
  • Очень плохое: все параметры выходят за пределы нормы.

Нормы:

  • Уровень загрязнения воздуха: до 50 мг/м³.
  • Уровень шума: до 70 дБ.
  • Концентрация углекислого газа: до 400 ppm.
  • Уровень радиации: до 0.3 мкЗв/ч.

Teston Lebra написал программу, которая запрашивает с консоли зарплату и выводит в консоль сколько останется денег после выплат налогов, а также информацию о страховании.

Условия по зарплате:

  • Зарплата от 400: вычитается 5% из зарплаты, медицинское и социальное страхование оплачивает компания.
  • Зарплата от 1200: вычитается 10% из зарплаты, медицинское страхование оплачивает компания, а социальное сам работник.
  • Зарплата от 2000: вычитается 15% из зарплаты, медицинское и социальное страхование оплачивает сам работник.

Программа хоть и работает, но не совсем корректно. Исправьте программу на более правильный вариант, чтобы она работала верно, и рядом напишите комментарии о том, почему изменённый вариант должен работать верно.

Код:

salary = int(input("Введите зарплату: "))

SALARY_MIN = 400
SALARY_MID = 1200
SALARY_MAX = 2000

if salary >= SALARY_MIN:
    tax_percent = 5
    tax_coef = 1 - tax_percent / 100
    salary = salary * tax_coef
    has_covered_med_tax = True
    has_covered_soc_tax = True
if salary >= SALARY_MID:
    tax_percent = 10
    tax_coef = 1 - tax_percent / 100
    salary = salary * tax_coef
    has_covered_med_tax = True
    has_covered_soc_tax = False
if salary >= SALARY_MAX:
    tax_percent = 15
    tax_coef = 1 - tax_percent / 100
    salary = salary * tax_coef
    has_covered_med_tax = False
    has_covered_soc_tax = False

print("Чистая прибыль:", salary)

if has_covered_med_tax == True:
    print("Компания покрывает медицинское страхование.")
if has_covered_soc_tax == True:
    print("Компания покрывает социальное страхование.")
