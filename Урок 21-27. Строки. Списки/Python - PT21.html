Практическое Задание 21.1

Напишите программу, которая выводит в консоль слово "Python", а затем выводит в консоль символы на чётных индексах.

Пример работы в консоли:

Python
P
t
o

Практическое Задание 21.2

Напишите программу, которая запрашивает с консоли строку, а затем выводит в консоль символы на чётных индексах в одну строку.

Пример работы в консоли:

Введите строку: Test String

T s  t i g

Практическое Задание 21.3

Напишите программу, которая запрашивает с консоли строку, формирует обратную строку и выводит в консоль результат.

Пример работы в консоли:

Введите строку: Test String

gnirtS tesT

Практическое Задание 21.4

Напишите программу, которая запрашивает с консоли слово маленькими буквами и выводит в консоль "Это палиндром." или "Это не палиндром." в зависимости от слова.

Пример работы в консоли:

Введите слово: потоп

Это палиндром.

Практическое Задание 21.5

Напишите программу, которая запрашивает с консоли строку и выводит в консоль последние 3 символа строки.

Пример работы в консоли:

Введите строку: abcdef

Последние 3 символа: f, e, d

Практическое Задание 21.6

Напишите программу, которая запрашивает с консоли строку и выводит в консоль каждый символ и следующий за ним символ в строке.

Пример работы в консоли:

Введите строку: Test String

Пары следующие: Te es st t   S St tr ri in ng g

Практическое Задание 21.7

Напишите программу, которая запрашивает с консоли фамилию, имя и отчество, формирует из первых букв новую строку, а затем выводит в консоль полученную строку.

Пример работы в консоли:

Введите фамилию: Lebra
Введите имя: Teston
Введите отчество: Experimenton

Инициалы: LTE

Практическое Задание 21.8

Напишите программу, которая запрашивает с консоли строку и символ, затем выводит в консоль количество вхождения символа в строку.

Пример работы в консоли:

Введите строку: Test String
Введите символ: t

Количество символа t в ведённой строке: 2

Практическое Задание 21.9

Напишите программу, которая запрашивает с консоли в одной строке дату и выводит в консоль данные на разных строчках.

Пример работы в консоли:

Введите дату (dd/mm/yyyy): 10/12/2023

День: 10
Месяц: 12
Год: 2023

Практическое Задание 21.10

Напишите программу, которая запрашивает с консоли 16-значный номер карты и выводит в консоль этот же номер, но с замаскированными символами, кроме последних четырёх.

Пример работы в консоли:

Введите номер карты: 5869586958695869

Ваш номер карты: ************5869
