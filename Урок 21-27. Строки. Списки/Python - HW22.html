Домашнее Задание 22.4

Для подсчёта точек проверяйте сразу две позиции. Например, если по индексу i находится точка, то проверяем по индексу i + 1. Если и на ней находится точка, то ничего не делаем. В противном случае увеличиваем счётчик предложений. Учтите, что в этом случае нужно проверять с индекса 0 до {размера_строки} - 1.

Пример работы в консоли:

Введите текст: abc? defg! hij... klm.

Количество предложений: 4

Домашнее Задание 22.8

  1. Генерируйте случайное число от 0 до размера набора символов. Полученное число будет индексом символа из набора символов. Используйте следующий код для генерации числа:
    from random import randint  # Подключение модуля random и импорт функции randint

index = randint(start, end)  # Использование функции для генерации одного числа от start до end включительно

Если пароль должен содержать цифры, то добавляйте к нему строку "0123456789". Если заглавные английские буквы, то "ABCDEFGHIJKLMNOPQRSTUVWXYZ". Если специальные символы, то "!@$%&*_-+=?".

Пароль должен содержать заглавные английские буквы (Y/N): Y
Пароль должен содержать специальные символы (Y/N): Y

Домашнее Задание 22.9 (Необязательное)

Teston Lebra попросили определить "текст" молекул ДНК по хешам. Напишите программу, которая перебирает все возможные комбинации молекул ДНК, вычисляет их хеши и выводит в консоль найденные последовательности ДНК.

Хеши следующие:
"fbfe4047a78675e30b3e6664392e308c83d5b7692180048db3652abad7cd19d4"
"b10ca0c7df6eea471a995c343bbf6acecb979f7fa9f91b6ce7f4e6b0825e1825"
"6037fa5f8f5312a8cca2a0a8a5f88395531e84d82deff50bae3f4e46360bfbc2"
"774ef38ec143565fcc12bad7161b596d2ede50110333857e777a2ac3fce07e9c"
"23f3b5a428129da00e3d3cb03812318104c9ccc8fe7562974719ec63db85f39e"

Известно, что размер "текста" молекул ДНК составляет 10 символов и этот "текст" состоит из заглавных символов "А", "Г", "Ц" и "Т". Программа должна выводить в формате {хеш} = {текст}.

Подсказки:

  1. Вводить ничего не нужно. Ваша программа должна перебирать возможные комбинации и определять их хеши. Если хеши совпадают с искомыми, то нужно выводить последовательность ДНК.
  2. Создайте переменную, которая будет хранить набор символов "АГЦТ".
  3. Создайте 5 переменных для каждого хеша.
  4. По структуре программа должна состоять из цикла while, внутри которого находится цикл for, внутри которого находится цикл for, внутри которого находится цикл for и т.д. В итоге должен получится внешний цикл while и 10 вложенных циклов for по принципу матрёшки. Цикл while должен прекратиться, когда было обнаружено 5 хешей. Циклы for должны перебирать числа от 0 до 3 включительно, чтобы с помощью этих чисел перебирать символы из "АГЦТ".
  5. В самом начале цикла while обнуляйте строку.
  6. Каждый следующий цикл for должен брать копию строки от предыдущего цикла и добавлять к копии символ из "АГЦТ":
    while ...:
    dna_str = ""
    for ... in range(...):
        dna_first = dna_str
        ...
        for ... in range(...):
            dna_second = dna_first
            ...
            for third_symb in range(4):
                dna_third = dna_second
                ...
  7. Для вычисления хеша используйте следующий код:
    from hashlib import sha256  # Подключение модуля hashlib и импорт функции sha256

dna_test = "АГТЦАГТЦАГ"  # Исходный текст
dna_test_utf_8 = dna_test.encode("utf-8")  # Кодируем данные в utf-8
dna_hash_obj = sha256(dna_test_utf_8)  # Создаём объект, в котором будет хэш данных
dna_hash = dna_hash_obj.hexdigest()  # Преобразуем объект-хэш в шестнадцатеричный формат

print(dna_hash)  # Для проверки работы
