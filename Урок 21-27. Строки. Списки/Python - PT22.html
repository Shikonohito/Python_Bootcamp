Практическое Задание 22.1

Напишите программу, которая реализует понятие "Срезы" для строк.

Консоль:

Введите строку: Test String
Введите начало среза: 0
Введите конец среза: 7
Введите шаг среза: 1

Полученный срез: Test S

Практическое Задание 22.2

Напишите программу, которая запрашивает с консоли строку и символ, который нужно удалить. Затем программа выводит в консоль исходную строку с удалёнными символами.

Консоль:

Введите строку: Test String
Введите строку: abcabcdabc abcd
Введите символ для удаления: d

Строка с удалённым символом d: abcabcabc abc

Практическое Задание 22.3

Напишите программу, которая запрашивает с консоли строку и символ, который нужно заменить. Затем запрашивает с консоли символ замены. После этого программа выводит в консоль исходную строку с заменёнными символами.

Консоль:

Введите строку: abcabcdabc abcd
Введите заменяемый символ: d
Введите заменяющий символ: *

Строка с заменёнными символами d на *: abcabc*abc abc*

Практическое Задание 22.4

Напишите программу, которая запрашивает с консоли строку и символ, который нужно удалить. Затем программа выводит в консоль исходную строку с удалённым вторым по счёту символом.

Консоль:

Введите строку: abcabcdabc abcd
Введите символ для удаления: d

Строка с удалённым символом d: abcabcdabc abc

Практическое Задание 22.5

Напишите программу, которая запрашивает с консоли строку и символ для поиска. Программа должна найти все вхождения символа и добавить их индексы в новую строку, затем вывести в консоль.

Консоль:

Введите строку: abcabcdabc abcd
Введите символ поиска: a

Найденные индексы: 0.3.7.11.
