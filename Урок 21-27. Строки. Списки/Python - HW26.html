Домашнее Задание 26.1

Teston Lebra услышал в новостях, что средняя зарплата по его стране выросла. Он посмеялся и пошёл вручную вычислять среднюю зарплату сотрудников в его работе. Напишите программу, которая запрашивает с консоли на каждой строке зарплату сотрудника до тех пор, пока не будет введена пустая строка. Введённые данные должны помещаться в список. Затем программа выводит в консоль среднюю арифметическую, медиану и моду по зарплатам.

Подсказки:

  1. После прочтения данных приведите их к числам.
  2. Медиана – это число, слева и справа от которого равное количество чисел из выборки.
  3. Для определения медианы сперва используйте метод списка sort(), чтобы отсортировать список. Затем следуйте условиям:
    — Если количество элементов в списке чётное, то нужно поделить количество на 2, взять число по полученному индексу и взять число на единицу меньше полученного индекса, сложить их и поделить на 2.
    Например:
    300 350 360 380 390 410 1100 2000 8000 10000
    10 чисел делим на 2 – получаем индекс 5. По этому индексу находится число 410. Берём ещё число на единицу меньше полученного индекса – это 390. Получаем медиану (390 + 410) / 2 = 400.

    — Если количество элементов в списке нечётное, то берём индекс последнего числа и делим на 2. Число по полученному индексу будет медианой.
    Например:
    300 350 360 380 390 410 1100 2000 8000 10000 999999
    11 чисел, индекс последнего числа 10. Делим 10 на 2 – получаем 5. По полученному индексу находится медиана – 410.
  4. Мода – это число, которое чаще всех повторяется в выборке. Если каждое число повторяется только 1 раз, то моды нет.
  5. Для вычисления моды сперва отсортируйте список при помощи его метода sort() для простоты алгоритма. Затем создайте переменную, которая будет хранить число, и переменную, которая будет хранить максимальное количество таких чисел (изначально 0). Также создайте переменную, которая будет хранить временное число, и переменную, которая будет хранить его количество.
  6. Во время прохода по списку, если рассматриваемое число не совпадает с числом из временной переменной, то проводите сравнение количества.
  7. Выводить нужно первую моду. Например:
    300 300 360 360 390 410 1100 2000 8000 10000
    Здесь две моды – 300 и 360, так как они обе повторяются по 2 раза. Выводить нужно первую моду, то есть 300.

Пример работы в консоли:

Введите зарплату сотрудника (1): 10000
Введите зарплату сотрудника (2): 8000
Введите зарплату сотрудника (3): 300
Введите зарплату сотрудника (4): 350
Введите зарплату сотрудника (5): 2000
Введите зарплату сотрудника (6): 1100
Введите зарплату сотрудника (7): 360
Введите зарплату сотрудника (8): 380
Введите зарплату сотрудника (9): 390
Введите зарплату сотрудника (10): 410
Введите зарплату сотрудника (11): 

Средняя зарплата по данным 10 сотрудников: 2329
Медианная зарплата по данным 10 сотрудников: 400
Мода зарплаты по данным 10 сотрудников: Отсутствует

Домашнее Задание 26.2

Реализуйте следующее меню: 

1. Зарегистрировать аккаунт
2. Войти
3. Завершить программу

В программе должны присутствовать два списка – список логинов и список паролей.

Выбран пункт 1:

  • Программа запрашивает с консоли новый логин и пароль.
  • Если введённый логин не совпадает ни с одним из логинов из списка логинов, то логин добавляется в список логинов и пароль добавляется в список паролей.
  • Логины и соответствующие пароли должны находиться под одинаковыми индексами. Например, если логин находится в списке логинов под индексом 3, то пароль от этого логина находится в списке паролей тоже под индексом 3.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Выбран пункт 2:

  • Программа запрашивает с консоли логин и пароль.
  • Если введённый логин совпадает с одним из логинов из списка логинов, а пароль совпадает с паролем из списка под тем же индексом, что и логин, то выводит главное меню.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Главное меню

1. Удалить текущий аккаунт
2. Изменить логин
3. Изменить пароль
4. Выйти из аккаунта
5. Показать логины и пароли всех аккаунтов
6. Завершить программу

Выбран пункт 1:

  • Программа запрашивает с консоли подтверждение удаления.
  • Если пользователь подтвердил, то логин и соответствующий пароль, через которые заходили в текущий аккаунт, должны быть удалены из списков, а программа возвращается в меню входа.

Выбран пункт 2:

  • Программа запрашивает с консоли логин.
  • Если введённый логин не совпадает ни с одним из логинов из списка логинов, то текущий логин заменяется на новый логин в списке логинов.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Выбран пункт 3:

  • Программа запрашивает с консоли пароль.
  • Программа заменяет текущий пароль на новый пароль в списке паролей.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Выбран пункт 4:

  • Программа возвращается в меню входа.

Выбран пункт 5:

  • Если у текущего аккаунта логин – "admin", а пароль – "5869", то программа выводит в консоль данные всех аккаунтов в виде: 
    {логин} - {пароль}
{логин} - {пароль}
{логин} - {пароль}
...
  • В противном случае программа выводит в консоль сообщение "Текущий аккаунт не обладает требуемыми правами.".

Затем программа выводит в консоль полученный список, после чего удаляет из списка имена и выводит в консоль изменённый список.

Подсказки:

Для удаления используйте del.

Введите имя: Kate
Введите имя: Jack

Введите имя: Jim

Домашнее Задание 26.4

Реализуйте следующее меню: 

1. Показать список
2. Ввести новый список имён
3. Добавить имя
4. Удалить имя по индексу
5. Изменить имя по индексу
6. Получить индекс по имени
7. Получить список имён по первой букве

В программе должен присутствовать списка имён.

Выбран пункт 1:

  • Программа выводит в консоль все имена в виде: 
    {имя} {имя} {имя} ...

Выбран пункт 2:

  • Программа запрашивает с консоли имена до тех пор, пока не введут пустую строку.
  • Из введённых данных программа формирует новый список и заменяет им основной список имён.

Выбран пункт 3:

  • Программа запрашивает с консоли имена до тех пор, пока не введут пустую строку.
  • Введённые данные программа добавляет в основной список имён.

Выбран пункт 4:

  • Программа запрашивает с консоли индекс.
  • По введённому индексу удаляет имя из основного списка имён. Используйте del.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Выбран пункт 5:

  • Программа запрашивает с консоли индекс.
  • По введённому индексу выводит в консоль имя.
  • Программа запрашивает с консоли имя.
  • Программа запрашивает с подтверждение на изменение.
  • Если пользователь подтвердил, то заменяет имя по индексу в списке имён на введённое.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Выбран пункт 6:

  • Программа запрашивает с консоли имя.
  • Если имя найдено в списке имён, то выводит в консоль индекс найденного имени.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Выбран пункт 7:

  • Программа запрашивает с консоли букву.
  • Если в списке имён найдено имя, которое начинается с введённой буквы, то добавляет имя во временный список.
  • Если временный список не пустой, то выводит в консоль имена из временного списка в виде:. 
    {имя} {имя} {имя} ...
  • В зависимости от результата в консоли выводится соответствующее сообщение.
