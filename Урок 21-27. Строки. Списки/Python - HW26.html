Домашнее Задание 26.1

Teston Lebra услышал в новостях, что средняя зарплата по его стране выросла. Он посмеялся и пошёл вручную вычислять среднюю зарплату сотрудников в его работе. Напишите программу, которая запрашивает с консоли на каждой строке зарплату сотрудника до тех пор, пока не будет введена пустая строка. Введённые данные должны помещаться в список. Затем программа выводит в консоль среднюю арифметическую, медиану и моду по зарплатам.

Подсказки:

  1. После прочтения данных приведите их к числам.
  2. Медиана - это число, слева и справа от которого равное количество чисел из выборки.
  3. Для определения медианы сперва используйте метод списка sort(), чтобы отсортировать список. Затем следуйте следующим условиям:
    — Если количество элементов в списке чётное, то нужно поделить количество на 2, взять число по полученному индексу и взять число на единицу меньше полученного индекса, сложить их и поделить на 2.
    Например:
    300 350 360 380 390 410 1100 2000 8000 10000
    10 чисел делим на 2 - получаем индекс 5. По этому индексу находится число 410. Берём ещё число на единицу меньше полученного индекса - это 390. Получаем медиану (390 + 410) / 2 = 400.

    — Если количество элементов в списке нечётное, то берём индекс последнего числа и делим на 2. Число по полученному индексу будет медианой.
    Например:
    300 350 360 380 390 410 1100 2000 8000 10000 999999
    11 чисел, индекс последнего числа 10. Делим 10 на 2 - получаем 5. По полученному индексу находится медиана - 410.
  4. Мода - это число, которое чаще всех повторяется в выборке. Если каждое число повторяется только 1 раз, то моды нет.
  5. Для вычисления моды сперва отсортируйте список при помощи его метода sort() для простоты алгоритма. Затем создайте переменную, которая будет хранить число, и переменную, которая будет хранить максимальное количество таких чисел (изначально 0). Также создайте переменную, которая будет хранить временное число, и переменную, которая будет хранить его количество.
  6. Во время прохода по списку, если рассматриваемое число не совпадает с числом из временной переменной, то проводите сравнение количества.
  7. Выводить нужно первую моду. Например:
    300 300 360 360 390 410 1100 2000 8000 10000
    Здесь две моды - 300 и 360, так как они обе повторяются по 2 раза. Выводить нужно первую моду, то есть 300.

Консоль:

Введите зарплату сотрудника (1): 10000
Введите зарплату сотрудника (2): 8000
Введите зарплату сотрудника (3): 300
Введите зарплату сотрудника (4): 350
Введите зарплату сотрудника (5): 2000
Введите зарплату сотрудника (6): 1100
Введите зарплату сотрудника (7): 360
Введите зарплату сотрудника (8): 380
Введите зарплату сотрудника (9): 390
Введите зарплату сотрудника (10): 410
Введите зарплату сотрудника (11): 

Средняя зарплата по данным 10 сотрудников: 2329
Медианная зарплата по данным 10 сотрудников: 400
Мода зарплаты по данным 10 сотрудников: Отсутствует

Домашнее Задание 26.2

Реализуйте следующее меню:
1. Зарегистрировать аккаунт
2. Войти
3. Завершить программу

При выборе пункта 1 программа запрашивает с консоли новый логин и пароль. Логин должен попасть в первый список, пароль во второй. Если введённый логин уже присутствует в списке, то программа должна сообщить об этом и снова запросить логин. После этого программа возвращается в меню входа. Организация списков следующая: первый список содержит только логины, а второй список только пароли. Если обратиться по индексу 0 у первого списка, то пароль от полученного логина тоже должен быть по индексу 0 у второго списка.

При выборе пункта 2 программа запрашивает с консоли логин и пароль. Программа должна позволить использовать любой зарегистрированный логин и соответствующий пароль. Если вход успешен, то программа выводит следующее меню:
1. Удалить текущий аккаунт
2. Изменить логин
3. Изменить пароль
4. Выйти из аккаунта
5. Показать логины и пароли всех аккаунтов
6. Завершить программу

При выборе пункта 1 программа должна запросить с консоли подтверждение удаления. В случае подтверждения, логин и пароль, с помощью которых был произведён вход, должны быть удалены из списков, и программа должна вернуться в меню входа.

При выборе пункта 2 программа должна запросить с консоли логин и внести изменения в список логинов.

При выборе пункта 3 программа должна запросить с консоли пароль и внести изменения в список паролей.

При выборе пункта 4 программа должна просто вернуться в меню входа.

При выборе пункта 5 программа должна показать в консоли все логины и пароли в формате "логин - пароль". Этот пункт должен работать, если зашли в аккаунт только с логином "admin" и паролем "5869".

Домашнее Задание 26.3

В месте, где работает Teston Lebra, начали сокращение сотрудников. Сотрудники, чьи имена находятся на нечётных индексах, будут увольнять. Напишите программу, которая запрашивает с консоли имена до тех пор, пока не будет введена пустая строка. Введённые данные должны помещаться в список. Затем программа выводит в консоль полученный список, после чего удаляет из списка имена и выводит в консоль изменённый список.

Подсказки:

Для удаления используйте del.

Консоль:

Введите имя: Kate
Введите имя: Jack
Введите имя: Bob
Введите имя: Jim
Введите имя: Teston
Введите имя: Arthur

Имена сотрудников: Kate, Jack, Bob, Jim, Teston, Arthur
Оставшиеся сотрудники: Kate, Bob, Teston

Домашнее Задание 26.4

Реализуйте следующее меню:
1. Показать список
2. Ввести новый список студентов
3. Добавить студентов
4. Удалить студента по индексу
5. Изменить студента по индексу
6. Получить индекс по имени
7. Получить список студентов по первой букве

При выборе пункта 1 программа показывает в консоли всех студентов.

При выборе пункта 2 программа запрашивает с консоли имена до тех пор, пока не введут пустую строку. Из введённых данных программа должна cформировать список.

При выборе пункта 3 программа запрашивает с консоли имена до тех пор, пока не введут пустую строку. Введённые данные программа должна добавлять список.

При выборе пункта 4 программа запрашивает с консоли индекс и удаляет студента по индексу. Используйте del.

При выборе пункта 5 программа запрашивает с консоли индекс, затем показывает в консоли имя под выбранным индексом. После этого программа запрашивает с консоли новое имя и запрашивает подтверждение на изменение.

При выборе пункта 6 программа запрашивает с консоли индекс, затем показывает в консоли имя под выбранным индексом.

При выборе пункта 7 программа запрашивает с консоли букву, затем проходит по списку имён и добавляет в новый список имена, которые начинаются на указанную букву (заглавные и строчные буквы программа должна считать за одинаковые буквы). После этого программа должна вывести в консоль имена из нового списка, если они есть. В противном случае программа должна вывести в консоль "Не найдено.".
