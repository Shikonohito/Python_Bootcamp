Практическое Задание 26.1

Реализуйте следующее меню: 

1. Показать список
2. Ввести новый список оценок
3. Добавить оценки
4. Удалить оценку по индексу
5. Изменить оценку по индексу
6. Определить количество оценок
7. Определить среднюю оценку
8. Определить медианную оценку
9. Завершить программу

Выбран пункт 1:

  • Программа выводит в консоль данные из списка в виде: 
    {оценка} {оценка} {оценка} ...

Выбран пункт 2:

  • Программа запрашивает с консоли оценки до тех пор, пока не введут пустую строку.
  • Из введённых чисел формируется новый список и основной список заменяется новым списком.

Выбран пункт 3:

  • Программа запрашивает с консоли оценки до тех пор, пока не введут пустую строку.
  • Введённые числа программа добавляет в основной список.

Выбран пункт 4:

  • Программа запрашивает с консоли индекс.
  • Программа удаляет по индексу оценку из списка.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Выбран пункт 5:

  • Программа запрашивает с консоли индекс.
  • Выводит в консоль оценку под указанным индексом.
  • Запрашивает с консоли новую оценку.
  • Если оценка от 1 до 12 включительно, то программа запрашивает подтверждение на изменение.
  • Если пользователь подтвердил, то оценка по указанному индексу заменяется на новую.
  • В зависимости от результата в консоли выводится соответствующее сообщение.

Выбран пункт 6:

  • Программа запрашивает с консоли оценку.
  • Выводит в консоль количество указанной оценки в списке.

Выбран пункт 7:

  • Программа выводит в консоль среднюю арифметическую оценку.

Выбран пункт 8:

  • Программа делает копию списка оценок.
  • Сортирует копию списка оценок.
  • Выводит в консоль среднюю медианную оценку.
