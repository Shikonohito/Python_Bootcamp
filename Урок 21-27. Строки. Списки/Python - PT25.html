Практическое Задание 25.1

Напишите программу, которая запрашивает с консоли температуры в одной строке и выводит в консоль среднее значение и список из введённых температур.

Пример работы в консоли:

Введите температуры через пробел: 25 26 30 29 30

Средняя температура: 28.0
Введённые температуры: 25 26 30 29 30

Практическое Задание 25.2

Напишите программу, которая запрашивает с консоли оценки до тех пор, пока не введут пустую строку. Затем программа формирует список и выводит в консоль минимальную оценку и список оценок.

Пример работы в консоли:

Введите оценку (1): 12
Введите оценку (2): 11
Введите оценку (3): 12
Введите оценку (4): 10
Введите оценку (5): 11
Введите оценку (6): 

Минимальная оценка: 10
Введённые оценки: 12 11 12 10 11

Практическое Задание 25.3

Реализуйте следующее меню: 

1. Показать список заказанного
2. Добавить новый заказ в конец
3. Добавить новый заказ в начало
4. Добавить новый заказ по номеру
5. Опустошить список заказов
6. Завершить программу

Выбран пункт 1:

  • Программа выводит в консоль каждый заказ в виде:
    • {порядковый_номер_заказа} - {название_заказанного}
{порядковый_номер_заказа} - {название_заказанного}
{порядковый_номер_заказа} - {название_заказанного}
...

Выбран пункт 2:

  • Программа запрашивает с консоли название заказа.
  • Добавляет введённое название в конец списка.

Выбран пункт 3:

  • Программа запрашивает с консоли название заказа.
  • Добавляет введённое название в начало списка.

Выбран пункт 4:

  • Программа запрашивает с консоли название заказа, затем запрашивает с консоли порядковый номер заказа.
  • Добавляет в список введённое название по указанному порядковому номеру.

Выбран пункт 5:

  • Программа запрашивает с консоли подтверждение.
  • Если пользователь подтвердил, то список заказов опустошается.

Практическое Задание 25.6

Даны 3 списка – список названий, список соответствующих цен и список с соответствующим количеством: 

product_name = ["Молоко", "Хлеб", "Сыр", "Масло", "Кофе", "Чай", "Сахар", "Мука", "Яйца", "Рис"]
product_price = [85, 40, 320, 150, 480, 300, 95, 60, 110, 130]
product_quantity = [20, 35, 0, 0, 5, 12, 25, 18, 30, 16]

Реализуйте следующее меню: 

1. Показать список товаров
2. Добавить товар
3. Удалить товар
4. Изменить товар
5. Продать товар
6. Завершить программу

Выбран пункт 1:

  • Программа выводит в консоль данные товаров в виде:
    • {название} - {цена} - {количество}
{название} - {цена} - {количество}
{название} - {цена} - {количество}
...

Выбран пункт 2:

  • Программа запрашивает с консоли название.
  • Если название найдено в списке названий, то запрашивает с консоли только количество и добавляет количество соответствующему продукту.
  • В противном случае, запрашивает с консоли цену и количество, затем добавляет в конец данные продукта в соответствующие списки.
  • В зависимости от результата программа показывает сообщение.

Выбран пункт 3:

  • Программа запрашивает с консоли название.
  • Если название найдено в списке названий, то удаляет из всех списков данные о найденном продукте.
  • В зависимости от результата программа показывает сообщение.

Выбран пункт 4:

  • Программа запрашивает с консоли название.
  • Если название найдено в списке названий, то программа запрашивает новое название, цену и количество.
  • Если введённые данные пустые, то исходные данные не меняются.
  • В зависимости от результата программа показывает сообщение.

Выбран пункт 5:

  • Программа запрашивает с консоли название.
  • Если название найдено в списке названий, то программа запрашивает количество.
  • Если количества достаточно, то показывает итоговую сумму и отнимает количество соответствующего товара.
  • В зависимости от результата программа показывает сообщение.

Практическое Задание 25.7

Реализуйте следующее меню: 

1. Внести доход/расход в отчёт
2. Показать полную историю
3. Показать историю по границам
4. Завершить программу

Выбран пункт 1:

  • Программа запрашивает с консоли числа до тех пор, пока не введут пустую строку.
  • Все числа помещаются в список.

Выбран пункт 2:

  • Программа выводит в консоль данные из списка в виде: 
    {индекс} {слово_доход/расход} {абсолютное_значение_числа}
{индекс} {слово_доход/расход} {абсолютное_значение_числа}
{индекс} {слово_доход/расход} {абсолютное_значение_числа}
...
  • Положительное число считается доходом, а отрицательное число – расходом.

Выбран пункт 3:

  • Программа запрашивает с консоли начальный индекс и конечный индекс.
  • По полученным границам программа выводит в консоль данные из списка в виде: 
    {индекс} {слово_доход/расход} {абсолютное_значение_числа}
{индекс} {слово_доход/расход} {абсолютное_значение_числа}
{индекс} {слово_доход/расход} {абсолютное_значение_числа}
...
  • Положительное число считается доходом, а отрицательное число – расходом.
