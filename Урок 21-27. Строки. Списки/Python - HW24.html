Домашнее Задание 24.6

Teston Lebra написал программу, которая ищет последовательность символов в строке и заменяет их на другую последовательность. Однако она работает некорректно. Исправьте программу так, чтобы она работала верно, и рядом напишите комментарии о том, почему изменённый вариант должен работать верно.

Код:

text = input("Введите строку: ")
to_find = input("Введите искомую строку: ")
to_change = input("Введите строку изменения: ")
len_text = len(text)
len_find = len(to_find)
result = ""

i = 0
while i < len_text:
    is_find = False
    j = 0
    while j < len_find:
        if text[i + j] == to_find[j]:
            is_find = True
            break
        j += 1
    if is_find:
        result += to_change
        i += len_find
    else:
        result += text[i]
        i += 1

print(f"Изменённая строка: {result}")

Консоль:

Введите строку: abcabcdabc abcd
Введите искомую строку: abcd
Введите строку изменения: xx

Изменённая строка: abcxxabc xx

  1. Создайте булеву переменную, по которой можно будет определить, нужно ли менять текущую букву на заглавную.

Консоль:
