Практическое Задание 27.5

Реализуйте следующее меню:
1. Поиск данных
2. Добавление данных
3. Удалить данные
4. Показать все данные
5. Завершить программу

При выборе пункта 1 программа показывает следующее:
1. Поиск по имени
2. Поиск по фамилии
3. Поиск по средней оценке
4. Вернуться
5. Завершить программу

При выборе пунктов 1-3 программа запрашивает с консоли имя/фамилию/среднюю оценку и указанные данные. Если находит, то все данные помещаются в новый список, а потом выводится в консоль. Если данные не найдены, то выводится в консоль "Не найдено.".

В главном меню при выборе пункта 2 программа запрашивает с консоли имя, фамилию и среднюю оценку на каждой строке. Затем формирует из этих данных список и помещает в общий список.

При выборе пункта 3 программа запрашивает с консоли индекс и удаляет по индексу.

При выборе пункта 4 программа выводит в консоль все данные.
