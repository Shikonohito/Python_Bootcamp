Практическое Задание 27.5

Реализуйте следующее меню: 

1. Показать все данные студента
2. Добавить нового студента
3. Удалить студента
4. Завершить программу

Студент представляется в программе в виде списка. Все студенты находятся в списке студентов. 

student_1 = ["Teston", "Lebra", 11.5]
student_2 = ["Tom", "Grid", 8.9]
student_3 = ["Kate", "Black", 11.2]
student_list = [student_1, student_2, student_3]

Выбран пункт 1:

  • Программа выводит в консоль данные из списка в виде: 
    {имя} {фамилия} {средняя_оценка}
{имя} {фамилия} {средняя_оценка}
{имя} {фамилия} {средняя_оценка}
...

Выбран пункт 2:

  • Программа запрашивает с консоли имя, фамилию и среднюю оценку на каждой строке.
  • Из введённых данных формируется новый студент и добавляется в основной список.

Выбран пункт 3:

  • Программа запрашивает с консоли имя и фамилию.
  • Если в основном списке найден студент с указанным именем и фамилией, то из основного списка удаляется найденный студент.
  • В зависимости от результата в консоли выводится соответствующее сообщение.
