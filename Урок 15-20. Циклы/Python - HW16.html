Домашнее Задание 16.7

Teston Lebra попросили доказать на практике, что шансы выпадения чисел 1, 2, 3, 4, 5 и 6 на игральной кости около 16.6%. Напишите программу, которая доказывает это при помощи симуляции.

Подсказки:

  1. Используйте следующий код, чтобы генерировать числа от 1 до 6 включительно:
    from random import randint  # Подключение модуля random и импорт функции randint

number = randint(1, 6)  # Генерация случайного числа от 1 до 6 включительно
  2. Программа не требует ввода пользователя.
  3. Для решения задания вам понадобится шесть переменных, в которых будут хранится количество выпадений чисел 1, 2, 3, 4, 5, 6, одна переменная для того, чтобы подсчитывать шаги цикла, одна переменная для хранения случайного числа и одна переменная для того, чтобы хранить количество бросков.
  4. Генерируйте числа внутри цикла при помощи randint(1, 6). Достаточно 1000000 бросков.
  5. Чтобы найти процент выпадения чисел, нужно количество выпадений числа поделить на количество бросков и умножить на 100. Например, если бросков 1000000 и количество выпадания числа 1 равно 166173, то получаем:
    166173 / 1000000 * 100 = 16.6173%

Домашнее Задание 16.8

Teston Lebra написал программу, которая запрашивает с консоли номер карты и выводит в консоль "Номер карты состоит только из чётных цифр." или "В номере карты присутствуют нечётные цифры." в зависимости от номера. Однако программа работает некорректно. Исправьте программу так, чтобы она работала верно, и рядом напишите комментарии о том, почему изменённый вариант должен работать верно.

Код:

card_num = int(input("Введите номер карты: "))
is_only_even = True

while card_num >= 0:
    digit = card_num % 10
    if digit % 2 != 0:
        is_only_even = False
        continue
    else:
        is_only_even = True

print()
if is_only_even:
    print("Номер карты состоит только из чётных цифр.")
else:
    print("В номере карты присутствуют нечётные цифры.")

Пример работы в консоли:

Введите номер карты: 5869586958695869

В номере карты присутствуют нечётные цифры.
