Практическое Задание 19.1

Реализуйте следующее меню:

1. Зарегистрироваться
2. Войти
3. Завершить программу

В программе есть две роли – клиент и администратор. Условимся, что на всю программу у нас может быть только один клиент и только один администратор.

Выбран пункт 1:

  • Программа запрашивает с консоли логин, пароль, имя, фамилию, 16-значный номер карты и CVV/CVC код.
  • Заменяет все данные клиента на новые.

Выбран пункт 2:

  • Программа запрашивает с консоли логин и пароль.
  • Если логин и пароль от клиента, то показывается меню клиента.
  • Если логин и пароль от администратора, то показывается меню администратора.
  • В противном случае в консоль выводится сообщение о проблеме.

Меню клиента:

Пользователь: {логин}    Роль: Клиент
1. Мой кабинет
2. Магазин
3. Выйти из аккаунта
4. Завершить программу

Выбран пункт 1:

  • Программа выводит в консоль идентификатор клиента, имя, фамилию, текущий баланс аккаунта.
  • Выводит в консоль следующее меню: 
    Идентификатор: {идентификатор}
Имя: {имя}
Фамилия: {фамилия}
Текущий баланс: {баланс}
===============================================
1. Изменить имя
2. Изменить фамилию
3. Пополнить счёт
4. Изменить данные карты
5. Вернуться
6. Завершить программу
  • Выбран пункт 1 или 2:
    • Программа запрашивает с консоли соответствующие данные.
    • Заменяет соответствующие данные у клиента.
  • Выбран пункт 3:
    • Программа запрашивает с консоли 16-значный номер карты и CVV/CVC код.
    • Если введённые данные совпадают с данными клиента, то программа запрашивает с консоли сумму пополнения.
    • Если сумма положительная, то увеличивает баланс на сумму пополнения.
    • В зависимости от результата в консоли выводится соответствующее сообщение.
  • Выбран пункт 4:
    • Программа запрашивает с консоли 16-значный номер карты и CVV/CVC код.
    • Заменяет данные карты на введённые данные.
  • Выбран пункт 5:
    • Программа возвращается в меню клиента.

Выбран пункт 2:

  • Программа выводит в консоль следующие данные и меню магазина: 
    Ваш текущий баланс: {баланс}
====================================================
1. Вернуться
2. Завершить программу
====================================================
Выберите товары, которые хотите купить.
3. Хлеб - количество: 100, цена: 0.4
4. Молоко - количество: 0, цена: 2.67
5. Яйца (10шт) - количество: 30, цена: 2.49
Выберите действие: ...
  • Выбран пункт с товаром:
    • Программа запрашивает количество для выбранного товара.
    • Вычисляет итоговую стоимость и выводит в консоль полную информацию.
    • Запрашивает с консоли подтверждение на покупку.
    • Если пользователь подтвердил, товара на складе достаточно и на балансе достаточное количество денег, то программа снимает деньги с баланса, а также отнимает количество доступного товара.

Меню администратора:

Пользователь: {логин}    Роль: Администратор
1. Показать данные клиента
2. Магазин
3. Выйти из аккаунта
4. Завершить программу

Выбран пункт 1:

  • Программа запрашивает с консоли идентификатор клиента.
  • Если введённый идентификатор совпадает с идентификатором клиента, то программа выводит в консоль имя, фамилию, текущий баланс клиента.

Выбран пункт 2:

  • Программа выводит в консоль следующие данные и меню магазина: 
    1. Вернуться
2. Завершить программу
====================================================
Выберите товар, чтобы обновить его данные.
3. Хлеб - количество: 100, цена: 0.4
4. Молоко - количество: 0, цена: 2.67
5. Яйца (10шт) - количество: 30, цена: 2.49
Выберите действие: ...
  • Выбран пункт с товаром:
    • Программа делает копию данных выбранного товара.
    • Выводит в консоль данные выбранного товара и данные копии.
    • Выводит в консоль меню изменения товара: 
      СТАРЫЕ ДАННЫЕ: {название_товара} - количество: {количество}, цена: {цена}
НОВЫЕ ДАННЫЕ: {название_товара} - количество: {количество}, цена: {цена}
====================================================
1. Изменить название
2. Изменить количество
3. Изменить цену
4. Подтвердить изменения и вернуться
5. Вернуться
    • Пункты 1-3 меняют данные только у копии.
    • Пункт 4 проверяет, положительные ли количество и цена у копии и, если да, то меняет старые данные на изменённые.
