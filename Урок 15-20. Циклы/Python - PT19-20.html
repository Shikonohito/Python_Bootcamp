Практическое Задание 19.1

Реализуйте следующее меню:

1. Зарегистрироваться
2. Войти
3. Завершить программу
Выберите действие: ...

В программе есть две роли - клиент и администратор.

При выборе пункта 1 программа запрашивает логин и пароль нового клиента. Затем возвращается в меню входа.

При выборе пункта 2 программа запрашивает логин и пароль. Если вводятся данные клиента, то показывается меню клиента. Если вводятся данные администратора, то показывается меню администратора.

Меню клиента:

Пользователь: <логин>    Роль: Клиент
1. Мой кабинет
2. Магазин
3. Выйти из аккаунта
4. Завершить программу
Выберите действие: ...

При выборе пункта 1 программа показывает идентификатор клиента, имя, фамилию, текущий баланс аккаунта, а также показывает меню:

Идентификатор: <строка>
Имя: <строка>
Фамилия: <строка>
Текущий баланс: <число>
=============================
1. Изменить имя
2. Изменить фамилию
3. Пополнить счёт
4. Вернуться
5. Завершить программу
Выберите действие: ...

При выборе пункта 3 программа запрашивает 16-значный номер карты, месяц и год окончания, CVV/CVC код. Если они совпадают с определёнными данными, то программа запрашивает сумму. В противном случае программа возвращает в меню "Мой кабинет".

При выборе пункта 4 программа возвращается в меню клиента.

В меню клиента при выборе пункта 2 программа показывает следующие данные и меню:

Ваш текущий баланс: <число>
====================================================
1. Оплатить корзину
2. Вернуться
3. Завершить программу
====================================================
Выберите товары, которые хотите положить в корзину.
4. Хлеб - количество: 100, цена: 0.4
5. Молоко - количество: 0, цена: 2.67
6. Яйца (10шт) - количество: 30, цена: 2.49
====================================================
Итоговая сумма: <число>
Ваша корзина:
- <товар> - <количество>
- <товар> - <количество>
Выберите действие: ...

При выборе товара программа запрашивает количество.

При выборе пункта 1 программа показывает итоговую сумму и корзину клиента, затем просит подтверждение. Если пользователь подтвердил и на балансе достаточное количество денег, то программа снимает деньги с баланса, а также отнимает количество товаров, добавляет историю покупок, затем очищает корзину клиента.

Меню администратора:

Пользователь: <логин>    Роль: Администратор
1. Показать данные клиента
2. Магазин
3. Выйти из аккаунта
4. Завершить программу
Выберите действие: ...

При выборе пункта 1 программа запрашивает идентификатор клиента. Если он верный, то программа показывает имя, фамилию, текущий баланс, историю покупок выбранного клиента.

При выборе пункта 2 программа показывает следующие данные и меню:

1. Вернуться
2. Завершить программу
====================================================
Выберите товар, чтобы обновить его данные.
3. Хлеб - количество: 100, цена: 0.4
4. Молоко - количество: 0, цена: 2.67
5. Яйца (10шт) - количество: 30, цена: 2.49
Выберите действие: ...

При выборе товара программа запрашивает новое название, количество и цену. Если вводятся пустые данные, то соответствующая информация не меняется. После этого программа показывает старые данные и новые данные, затем запрашивает подтверждение на изменение.
