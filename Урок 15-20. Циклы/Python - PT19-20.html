Практическое Задание 19.1

Реализуйте следующее меню:

1. Зарегистрироваться
2. Войти
3. Завершить программу

В программе есть две роли - клиент и администратор. Условимся, что на всю программу у нас может быть только один клиент и только один администратор.

Выбор пункта 1:
— Программа запрашивает с консоли логин, пароль, имя, фамилию, 16-значный номер карты и CVV/CVC код.
— Программа заменяет данные старого клиента на новые данные.
— Возвращается в меню входа.

Выбор пункта 2:
— Программа запрашивает с консоли логин и пароль.
— Если вводятся данные клиента, то показывается меню клиента.
— Если вводятся данные администратора, то показывается меню администратора.

Меню клиента:

Пользователь: {логин}    Роль: Клиент
1. Мой кабинет
2. Магазин
3. Выйти из аккаунта
4. Завершить программу

В меню клиента выбор пункта 1:
— Программа выводит в консоль идентификатор клиента, имя, фамилию, текущий баланс аккаунта.
— Программа выводит в консоль меню: 

Идентификатор: {идентификатор}
Имя: {имя}
Фамилия: {фамилия}
Текущий баланс: {баланс}
=============================
1. Изменить имя
2. Изменить фамилию
3. Пополнить счёт
4. Изменить данные карты
5. Вернуться
6. Завершить программу

В меню кабинета выбор пункта 3:
— Программа запрашивает с консоли 16-значный номер карты и CVV/CVC код.
— Если они совпадают с данными клиента, то программа запрашивает с консоли сумму и добавляет на баланс карты. В противном случае программа возвращает в меню кабинета.

Выбор пункта 4:
— Программа запрашивает с консоли 16-значный номер карты и CVV/CVC код.
— Программа заменяет данные карты на введённые.

Выбор пункта 5:
— Программа возвращается в меню клиента.

В меню клиента выбор пункта 2:
— Программа выводит в консоль следующие данные и меню магазина: 

Ваш текущий баланс: {баланс}
====================================================
1. Вернуться
2. Завершить программу
====================================================
Выберите товары, которые хотите купить.
3. Хлеб - количество: 100, цена: 0.4
4. Молоко - количество: 0, цена: 2.67
5. Яйца (10шт) - количество: 30, цена: 2.49

Выбор товара:
— Программа запрашивает с консоли количество.
— Если товара достаточно, то уменьшает количество товара, вычисляет стоимость и вычитывает из текущего баланса карты.

Меню администратора:

Пользователь: {логин}    Роль: Администратор
1. Показать данные клиента
2. Магазин
3. Выйти из аккаунта
4. Завершить программу

Выбор пункта 1:
— Программа запрашивает с консоли идентификатор клиента.
— Если введённый идентификатор верный, то программа выводит в консоль имя, фамилию, текущий баланс.

Выбор пункта 2:
— Программа выводит в консоль следующие данные и меню: 

1. Вернуться
2. Завершить программу
====================================================
Выберите товар, чтобы обновить его данные.
3. Хлеб - количество: 100, цена: 0.4
4. Молоко - количество: 0, цена: 2.67
5. Яйца (10шт) - количество: 30, цена: 2.49

Выбор товара:
— Программа делает копию данных выбранного товара.
— Выводит в консоль данные выбранного товара и данные копии.
— Выводит в консоль меню изменения товара: 

СТАРЫЕ ДАННЫЕ: {название_товара} - количество: {количество}, цена: {цена}
НОВЫЕ ДАННЫЕ: {название_товара} - количество: {количество}, цена: {цена}
====================================================
1. Изменить название
2. Изменить количество
3. Изменить цену
4. Подтвердить изменения и вернуться
5. Вернуться

Пункты 1-3 меняют данные только у копии. При выборе пункта 4 программа проверяет, в порядке ли данные у копии и, если да, то меняет старые данные на изменённые.
