Практическое Задание 16.1

Напишите программу, которая запрашивает с консоли число и выводит в консоль первое число, которое делится на 7 нацело после введённого числа.

Пример работы в консоли:

Введите число: 14

Число 21 первое число, которое делится на 7 нацело.

Практическое Задание 16.2

Реализуйте следующее меню: 

1. Найти сумму чисел
2. Определить чётность числа
3. Завершить программу

Выбран пункт 1:

  • Программа запрашивает с консоли два числа.
  • Выводит в консоль сумму чисел.

Выбран пункт 2:

  • Программа запрашивает с консоли одно число.
  • Выводит в консоль чётное ли введённое число.

Выбран пункт 3:

  • Программа завершает работу.

Практическое Задание 16.3

Напишите программу, которая загадывает число от 1 до 100 включительно, затем запрашивает с консоли число. Если число угадано, то программа выводит в консоль количество попыток, затем запрашивает с консоли ввод на вопрос "Сыграть ещё раз (Y/N): ". Если ввели "Y", то игра начинается заново. В противном случае программа завершается. Если же число не угадано, то программа выводит в консоль "Загаданное число больше." или "Загаданное число меньше.".

Подсказка:

Используйте следующий код, чтобы генерировать число от 1 до 100 включительно: 

from random import randint  # Подключение модуля random и импорт функции randint

randint(1, 100)  # Использование функции для генерации одного числа от 1 до 100 включительно

Пример работы в консоли:

Введите загаданное число: 5
Загаданное число больше.

Введите загаданное число: 7
Загаданное число меньше.

Введите загаданное число: 6
=============================
Отгадано с 3 попытки.
Сыграть ещё раз (Y/N): N
Программа завершается.

Практическое Задание 16.4

Напишите программу, которая проверяет корректность IP. Программа запрашивает с консоли числа от 0 до 255 включительно. Если из 4-х запросов хотя бы одно число было вне диапазона, то программа выводит в консоль "Неверный IP адрес.". В противном случае выводит "IP адрес корректный.".

Пример работы в консоли:

Введите первую часть IP: 243
Введите вторую часть IP: 277

Неверный IP адрес.

Практическое Задание 16.5

Напишите программу, которая последовательно запрашивает с консоли стоимость корзины, скидку от 1 до 100 и расстояние от 0 до 20 включительно. Если какой-то запрос с консоли оказался неверным, то программа полностью заново запрашивает данные. Программа должна применить указанную скидку к стоимости корзины. Затем программа за каждый 1 км должна добавить к итоговой сумме 150.

Пример работы в консоли:

Стоимость корзины: -5000
================================
Указана неверная стоимость корзины.
Запрос перезапущен.
================================
Стоимость корзины: 5000
Введите скидку (%): 125
================================
Указана неверная скидка.
Запрос перезапущен.
================================
Стоимость корзины: 5000
Введите скидку (%): 50
Введите расстояние (км): 21
================================
Указано неверное расстояние.
Запрос перезапущен.
================================
Стоимость корзины: 5000
Введите скидку (%): 50
Введите расстояние (км): 10
================================
Итоговая стоимость: 4000
