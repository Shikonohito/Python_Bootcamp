Plugin - Presentation Assistant

Shift + F10 - Run
Ctrl + F5 - Rerun
Ctrl + Alt + S - Настройки
Ctrl + Alt + L - Реформатирование кода
Alt + 1 - Панель проекта
Alt + 4 - Панель запуска
Alt + 6 - Панель проблем
Alt + 7 - Панель структур
Alt + F12 - Панель терминала
Shift + F12 - Вернуть положение текущего окна как было изначально
Shift + F9 - Debug Run

Во время дебага:
Ctrl + F2 - Stop Run
F8 - Перешагнуть
F7 - Войти внутрь
Shift + F8 - Выйти
F9 - Resume Run

Ctrl + D - Дублировать строку
Ctrl + Y - Стереть строку
Ctrl + Z - Отменить последнее действие
Ctrl + Shift + Z - Вернуть последнее действие
Ctrl + X - Вырезать выделенное
Ctrl + C - Скопировать выделенное
Ctrl + V - Вставить
Ctrl + W - Выделение с расширением 
Ctrl + Shift + W - Выделение с сужением
Ctrl + Shift + U - Перевод выделенного в заглавные или в строчные буквы
Ctrl + Alt + Enter - Создание новой строки над текущей строкой
Ctrl + Shift + Enter - Создание новой строки под текущей строкой. Если текущая строка не закончена, то заканчивает её
Ctrl + Q, когда поставил курсор на название функции - Быстрая документация
Alt + Shift + Стрелочки - Перемещение строки вверх или вниз
Ctrl + F - Поиск в текущем файле
Ctrl + R - Поиск и замена в текущем файле
Ctrl + Shift + F - Поиск в проекте
Ctrl + Shift + R - Поиск и замена в проекте
Ctrl + Backspace, когда в поиске - Стереть слово
Ctrl + Tab - Переключение между вкладками
Alt + F7, когда поставил курсор на название функции - Показывает, где используется
Alt + Enter, когда поставил курсор на подчёркнутое - Показывает контекст и действия
Ctrl + / - Закомментировать или раскомментировать строку
Shift + F6, когда выделил файл  - Переименовать
Ctrl + E - Недавние файлы
Ctrl + Shift + E - Места, где были недавние изменения
Alt + Insert - Генератор кода
Ctrl + Alt + V - Быстрое использование и создание переменной
Ctrl + Alt + M, когда выделен блок - Быстрое использование и создание метода
Ctrl + Alt + N, когда курсор на переменной или методе - Быстрая вставка значений с возможным удалением
Shift + F6, когда курсор на названии - Быстрая смена всех таких же имён
Ctrl + Alt + P, когда курсор на значении в методе - Быстрое создание параметра
Shift x2 - Поиск
Ctrl + G - Переход на определённую строку и столбец
