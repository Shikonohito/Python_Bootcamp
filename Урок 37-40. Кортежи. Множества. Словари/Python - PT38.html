Практическое Задание 38.1

Реализуйте следующее меню: 

1. Показать логины и пароли пользователей
2. Зарегистрировать пользователя
3. Удалить пользователя
4. Изменить пароль пользователя
5. Завершить программу

Данные пользователей хранятся в виде одного словаря: 

{"login_1": "password_1", "login_2": "password_2", ...}

Выбран пункт 1:

  • Программа выводит в консоль данные в следующем виде: 
    {логин} - {пароль}
{логин} - {пароль}
...

Выбран пункт 2:

  • Программа запрашивает с консоли новые логин и пароль.
  • Если введённого логина нет в словаре, то в словарь добавляется логин с паролем.
  • В зависимости от результата программа выводит в консоль сообщение.

Выбран пункт 3:

  • Программа запрашивает с консоли логин.
  • Если введённый логин есть в словаре, то из словаря удаляется логин с паролем.
  • В зависимости от результата программа выводит в консоль сообщение.

Выбран пункт 4:

  • Программа запрашивает с консоли логин и новый пароль.
  • Если введённый логин есть в словаре, то у логина меняется пароль.
  • В зависимости от результата программа выводит в консоль сообщение.

Практическое Задание 38.2

Реализуйте следующее меню: 

1. Показать данные студента
2. Изменить Фамилию и Имя
3. Изменить дату рождения
4. Добавить оценку
5. Удалить оценку по порядковому номеру
6. Изменить оценку по порядковому номеру
7. Завершить программу

Студент хранится в виде одного словаря: 

{"f_name": "Teston", "l_name": "Lebra", "birth_date": 10/02/2000, "grades": [12, 12, 12]}

Дата рождения хранится как объект datetime.date().

Выбран пункт 1:

  • Программа выводит в консоль данные в следующем виде: 
    {имя} {фамилия}
Возраст: {возраст}
Оценки: {оценка} {оценка} {оценка} ...
  • Для вычисления возраста напишите функцию calculate_age, которая принимает день рождения и возвращает возраст. Используйте datetime.now(), чтобы получить текущую дату, и свойства day, month и year, чтобы получить день, месяц и год соответственно из даты.

Выбран пункт 2:

  • Программа запрашивает с консоли новые фамилию и имя.
  • Заменяет фамилию и имя в словаре.

Выбран пункт 3:

  • Программа запрашивает с консоли дату рождения в формате "%d/%m/%Y".
  • Заменяет дату рождения в словаре.
  • Используйте следующий код для сохранения объекта datetime: 
    from datetime import datetime  # Подключение модуля datetime и импорт класса datetime

# Метод strptime принимает дату и формат в виде строк, возвращает дату и время как объект datetime
# Метод date возвращает только дату
birth_date = datetime.strptime("10/02/2000", "%d/%m/%Y").date()

Выбран пункт 4:

  • Программа запрашивает с консоли новую оценку от 1 до 12 включительно.
  • Добавляет оценку студенту в список.

Выбран пункт 5:

  • Программа запрашивает с консоли порядковый номер оценки.
  • Удаляет оценку по номеру.

Выбран пункт 6:

  • Программа запрашивает с консоли порядковый номер оценки и новую оценку.
  • Заменяет заменяет у студента оценку по номеру.
