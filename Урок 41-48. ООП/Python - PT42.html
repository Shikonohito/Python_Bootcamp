Практическое Задание 42.1

Создайте класс "Студент". Этот класс должен хранить поля:

  • Идентификатор
  • Имя
  • Фамилия
  • Группа
  • Список оценок
  • Среднюю оценку

Для класса определите методы:

  • Конструктор (__init__)
  • Строковое представление (__str__)
  • Добавление оценки
    • Принимает оценку.
    • Если оценка от 1 до 12 включительно, то добавляет оценку в список. Обновляет среднюю оценку. Возвращает True.
    • В противном случае возвращает False.

Практическое Задание 42.2

Используя созданный класс в Практическом Задании 42.1, реализуйте следующее меню: 

1. Показать всех студентов
2. Показать студента по идентификатору
3. Добавить нового студента в список
4. Удалить студента из списка по идентификатору
5. Добавить оценку студенту

Все студенты хранятся в списке. Студент – это экземпляр класса "Студент".

Выбран пункт 1:

  • Программа проходит по каждому студенту из списка и выводит в консоль его строковое представление в виде: 
    {идентификатор} {имя} {фамилия} {группа} {средняя_оценка}

Выбран пункт 2:

  • Программа запрашивает с консоли идентификатор студента.
  • Если студент с указанным идентификатором найден в списке, то выводит в консоль его строковое представление и оценки в виде: 
    {идентификатор} {имя} {фамилия} {группа} {средняя_оценка}
{оценка} {оценка} {оценка} ...
  • В противном случае программа сообщает о результате.

Выбран пункт 3:

  • Программа запрашивает с консоли идентификатор, имя, фамилию, группу.
  • Если студент с указанным идентификатором не найден в списке, то добавляет его в список.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Выбран пункт 4:

  • Программа запрашивает с консоли идентификатор.
  • Если студент с указанным идентификатором найден в списке, то удаляет его из списка.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Выбран пункт 5:

  • Программа запрашивает с консоли идентификатор.
  • Если студент с указанным идентификатором найден в списке, то запрашивает с консоли оценку и добавляет её студенту.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Практическое Задание 42.3

Создайте класс "Банковская карта". Этот класс должен хранить поля:

  • Идентификатор
  • Имя
  • Фамилия
  • Баланс
  • CVV/CVC код

Для класса определите методы:

  • Конструктор (__init__)
  • Строковое представление (__str__)
  • Пополнение баланса
    • Принимает сумму пополнения.
    • Если сумма положительная, то добавляет сумму к балансу. Возвращает True.
    • В противном случае возвращает False.
  • Снятие с баланса
    • Принимает сумму снятия.
    • Если сумма положительная и баланс больше-равен сумме, то отнимает сумму из баланса. Возвращает True.
    • В противном случае возвращает False.

Практическое Задание 42.4

Используя созданный класс в Практическом Задании 42.3, реализуйте следующее меню: 

1. Показать все карты
2. Показать карту по идентификатору
3. Добавить карту в список
4. Удалить карту из списка по идентификатору
5. Пополнить баланс карты по идентификатору
6. Снять баланс карты по идентификатору

Все карты хранятся в списке. Карта – это экземпляр класса "Банковская карта".

Выбран пункт 1:

  • Программа проходит по каждой карте из списка и выводит в консоль её строковое представление в виде: 
    {идентификатор} {имя} {фамилия}

Выбран пункт 2:

  • Программа запрашивает с консоли идентификатор карты.
  • Если карта с указанным идентификатором найдена в списке, то запрашивает с консоли CVV/CVC код.
  • Если CVV/CVC код совпадает с указанной на карте, то выводит в консоль её строковое представление, баланс и CVV/CVC код карты в виде: 
    {идентификатор} {имя} {фамилия} {CVV/CVC_код}
Баланс: {баланс}
  • В противном случае программа сообщает о результате.

Выбран пункт 3:

  • Программа запрашивает с консоли идентификатор, имя, фамилию, CVV/CVC код, баланс.
  • Если карта с указанным идентификатором не найдена в списке, то добавляет её в список.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Выбран пункт 4:

  • Программа запрашивает с консоли идентификатор.
  • Если карта с указанным идентификатором найдена в списке, то удаляет её из списка.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Выбран пункт 5:

  • Программа запрашивает с консоли идентификатор.
  • Если карта с указанным идентификатором найдена, то запрашивает с консоли сумму пополнения и пополняет баланс карты.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Выбран пункт 6:

  • Программа запрашивает с консоли идентификатор.
  • Если карта с указанным идентификатором найдена, то запрашивает с консоли сумму снятия и снимает баланс карты.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.
