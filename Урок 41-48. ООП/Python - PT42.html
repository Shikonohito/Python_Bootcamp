Практическое Задание 42.1

Создайте класс "Студент". Этот класс должен хранить поля для идентификатора, имени, фамилии, группы, списка оценок и средней оценки.
Для класса определите строковое представление (__str__) и конструктор (__init__).
Для класса определите метод добавления оценки. Метод принимает целую оценку от 1 до 12 включительно.

Практическое Задание 42.2

Используя созданный класс в Практическом Задании 42.1, реализуйте следующее меню: 

1. Показать всех студентов
2. Показать студента по идентификатору
3. Добавить нового студента в список
4. Удалить студента из списка по идентификатору
5. Добавить оценку студенту

Практическое Задание 42.3

Создайте класс "Банковская карта". Этот класс должен хранить поля для идентификатора, имени, фамилии, баланса и CVV/CVC кода.
Для класса определите строковое представление (__str__) и конструктор (__init__).
Для класса определите метод проверки баланса. Метод принимает целое число.. Если принимаемое число больше нуля, то возвращает True, в противном случае возвращает False.
Для класса определите метод пополнения баланса. Метод принимает целое число, вызывает метод проверки баланса. Если метод вернул True, то пополняет баланс, возвращает True. В противном случае возвращает False.
Для класса определите метод снятия с баланса. Метод принимает целое число, вызывает метод проверки баланса. Если метод вернул True, то снимает с баланса, возвращает True. В противном случае возвращает False.

Практическое Задание 42.4

Используя созданный класс в Практическом Задании 42.3, реализуйте следующее меню: 

1. Показать все карты
2. Показать карту по идентификатору
3. Добавить карту в список
4. Удалить карту из списка по идентификатору
5. Пополнить баланс карты по идентификатору
6. Снять баланс карты по идентификатору
