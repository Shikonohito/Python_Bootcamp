Создайте класс "Плейлист". Этот класс должен хранить поля:

  • Название
  • Список песен

Для класса определите методы:

  • Конструктор
  • Строковое представление

Создайте по одному экземпляру каждого класса, добавьте их в список, а также создайте список из 3 элементов, для реализации утиной типизации.

Практическое Задание 46.3

Создайте класс "Метод оплаты". Этот класс должен хранить поля:

  • Идентификатор
  • Баланс

Для класса определите методы:

  • Конструктор
  • Строковое представление
  • Оплата
    • Принимает сумму оплаты.
    • Если сумма положительна и на балансе достаточно денег, то отнимает из баланса сумму. Возвращает True.
    • В противном случае возвращает True.

Создайте класс "Банковская карта", который наследуется от класса "Метод оплаты". Этот класс должен хранить поля:

  • Коэффициент налога

Для класса определите методы:

  • Конструктор
  • Строковое представление
  • Оплата
    • Принимает сумму оплаты.
    • Вызывает оплату родителя, передав сумму оплаты, умножив на коэффициент.
    • Возвращает то, что вернёт метод родителя.

Создайте класс "Бонусная карта", который наследуется от класса "Метод оплаты". Этот класс должен хранить поля:

    • Если сумма оплаты не превышает лимит оплаты, то вызывает оплату родителя, передав сумму оплаты, умножив на коэффициент.
    • Возвращает то, что вернёт метод родителя.

Создайте класс "Криптокошелёк", который наследуется от класса "Метод оплаты". Этот класс должен хранить поля:

Практическое Задание 46.4

Используя классы из Практического Задания 46.3, создайте по одному экземпляру каждого класса. Программа должна запросить с консоли сумму оплаты, затем предложить на выбор 3 варианта. В зависимости от выбора пользователя производится соответствующая оплата.

Реализуйте оплату через функцию, которая принимает метод оплаты и сумму, производит оплату и выводит результат работы.
