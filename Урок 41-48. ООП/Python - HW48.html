Домашнее Задание 48.1

Создайте базу данных клиентов и реализуйте CRUD. Сделайте так, чтобы клиенты могли иметь несколько электронных карточек для оплат услуг.

Клиентов нужно как-то идентифицировать. У них есть имя, фамилия и список электронных карт.

Электронные карты также нужно как-то идентифицировать. У них есть баланс и срок годности.

Необязательно: реализуйте подписки для клиентов. У клиента должна храниться дата окончания подписки. Подписка на 1 месяц стоит 5 единиц, на 3 месяца – 13 единиц, на 1 год – 49 единиц. Выдавать подписку должна база данных.

Подсказка:

Используйте следующий код для реализации срока годности электронных карт: 

from datetime import datetime, timedelta

now_date = datetime.now().date()  # Формирование текущей даты
after_5_days_from_now = now_date + timedelta(days=5)  # Добавление дней к дате
print(after_5_days_from_now)

card_date = datetime(2025, 12, 31).date()  # Формирование даты по данным
print(card_date >= now_date)  # Проверка, наступила ли дата сard_date. Будет False, если наступило
