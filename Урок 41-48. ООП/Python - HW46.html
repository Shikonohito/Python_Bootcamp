Домашнее Задание 46.1

Создайте класс "Игрок". Этот класс должен хранить поля:

  • Имя
  • Уровень
  • Текущий опыт
  • Требуемый опыт для следующего уровня

Для класса определите методы:

  • Конструктор
  • Строковое представление
  • Свойства
  • Повышение уровня
    • Ничего не принимает.
    • Обнуляет текущий опыт.
    • Увеличивает требуемый опыт для следующего уровня на 200.
    • Увеличивает уровень на 1.
  • Получение опыта
    • Принимает число.
    • Увеличивает текущий опыт на число.
    • Если текущий опыт больше-равен требуемого опыта для следующего уровня, то вызывает повышение уровня.

Создайте класс "Маг", который наследуется от класса "Игрок". Этот класс должен хранить поля:

  • Список изученных книг

Для класса определите методы:

  • Конструктор
  • Получение изученных книг
  • Изучение книги
    • Принимает название книги.
    • Добавляет название в список.
    • Вызывает получение опыта с числом 200.

Создайте класс "Вор", который наследуется от класса "Игрок". Этот класс должен хранить поля:

  • Деньги

Для класса определите методы:

  • Конструктор
  • Получение количества денег
  • Кража
    • Принимает количество денег.
    • Если денег до 200, то шанс украсть 100%.
    • Если денег от 200 до 500, то шанс украсть 75%.
    • Если денег от 500, то шанс украсть 50%.
    • Если украсть удалось, то увеличивает количество денег и вызывает получение опыта, передав количество денег делённые на 10. Возвращает True.
    • В противном случае возвращает False.

Подсказка:

Для того, чтобы узнать, получилось украсть или нет, взять случайное число и определить, входит ли он в нужный промежуток. Для генерации случайного числа используйте код, указанный ниже: 

from random import randint # Подключение модуля random и импорт функции randint

start = 1
end = 100
number = randint(start, end) # Генерация случайного числа от start до end

Домашнее Задание 46.2

Используя классы из Домашнего Задания 46.1, создайте по одному экземпляру каждого класса и реализуйте следующее меню: 

1. Выдать книгу магу
2. Указать цель для вора
3. Добавить всем опыта

Выбран пункт 1:

  • Программа запрашивает с консоли название книги.
  • У мага вызывается изучение книги.
  • Программа выводит в консоль строковое представление мага и список изученных книг.

Выбран пункт 2:

  • Программа запрашивает с консоли количество денег у цели.
  • У вора вызывается кража.
  • В зависимости от результата, программа выводит в консоль результат, строковое представление вора и количество денег.

Выбран пункт 3:

  • Программа вызывает функцию, передав ей список персонажей и количество опыта.
  • Функция выводит в консоль строковое представление каждого персонажа.
  • Вызывает у каждого персонажа получение опыта.
  • Снова выводит в консоль строковое представление каждого персонажа.

Домашнее Задание 46.4

Используя классы из Домашнего Задания 46.3, создайте по одному экземпляру каждого класса, поместите их в список и реализуйте следующее меню: 

1. Показать все товары
2. Добавить новый товар
3. Применить скидку к товару

Выбран пункт 1:

  • Программа проходит по каждому товару и выводит в консоль его строковое представление в виде: 
    {идентификатор} {цена} {действующая_скидка} {остальные_данные}
{идентификатор} {цена} {действующая_скидка} {остальные_данные}
...

Выбран пункт 2:

  • Программа запрашивает с консоли идентификатор, цену, тип товара.
  • В зависимости от типа товара программа запрашивает с консоли дополнительную информацию и создаёт соответствующий экземпляр класса.
  • Если товара с указанным идентификатором нет в списке, то добавляет товар в список.
  • В зависимости от результата, программа выводит в консоль результат.

Выбран пункт 3:

  • Программа запрашивает с консоли идентификатор и скидку.
  • Если товар с указанным идентификатором найден в списке, то вызывает применение скидки у товара.
  • В зависимости от результата, программа выводит в консоль результат.
