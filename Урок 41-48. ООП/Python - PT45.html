Практическое Задание 45.1

Создайте класс "Человек". Этот класс должен хранить поля:

  • Имя
  • Возраст

Для класса определите методы:

  • Конструктор
  • Строковое представление
  • Свойства

Создайте класс "Студент", который наследуется от класса "Человек". Этот класс должен хранить поля:

  • Оценки в виде словаря, значения которого список оценок, а ключи – название предмета

Для класса определите методы:

  • Конструктор
  • Добавление оценки
    • Принимает словарь, где ключ – название предмета, а значение – оценка.
    • Если ключ существует, то добавляет оценку по ключу в список.
    • Если ключ не существует, то добавляет ключ и список с указанной оценкой.
  • Получение средней оценки по предмету
    • Принимает название предмета.
    • Возвращает среднюю оценку по названию предмета.
  • Получение средней оценки
    • Ничего не принимает.
    • Возвращает среднюю оценку по всем предметам.
  • Получение всех названий предметов, по которым есть оценки
    • Ничего не принимает.
    • Возвращает кортеж названий предметов.

Создайте класс "Преподаватель", который наследуется от класса "Человек". Этот класс должен хранить поля:

  • Предмет, который ведёт преподаватель

Для класса определите методы:

  • Конструктор
  • Строковое представление
  • Свойства
  • Присвоение оценки студенту
    • Принимает студента и оценку.
    • Добавляет студенту оценку по своему предмету.

Практическое Задание 45.2

Используя классы из Практического Задания 45.1, реализуйте следующее меню: 

1. Добавить преподавателя
2. Добавить студента
3. Показать подробную информацию о преподавателе
4. Показать подробную информацию о студенте
5. Выдать оценку от преподавателя
6. Завершить программу

Все преподаватели хранятся в списке преподавателей. Преподаватель – это экземпляр класса "Преподаватель".

Все студенты хранятся в списке студентов. Студент – это экземпляр класса "Студент".

Выбран пункт 1:

  • Программа запрашивает с консоли имя, возраст и предмет.
  • Формирует из данных преподавателя.
  • Пытается добавить преподавателя в список преподавателей.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Выбран пункт 2:

  • Программа запрашивает с консоли имя и возраст.
  • Формирует из данных студента.
  • Пытается добавить студента в список студентов.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Выбран пункт 3:

  • Программа запрашивает с консоли имя.
  • Если преподаватель с указанным именем найден в списке преподавателей, то выводит в консоль его строковое представление в виде: 
    {имя} {возраст} {предмет}

Выбран пункт 4:

  • Программа запрашивает с консоли имя.
  • Если студент с указанным именем найден в списке студентов, то выводит в консоль его строковое представление, среднюю оценку, все оценки и среднюю оценку по предмету в виде: 
    {имя} {возраст} {общая_средняя_оценка}
{средняя_оценка_предмета} {название_предмета} {оценка} {оценка} {оценка} ...
{средняя_оценка_предмета} {название_предмета} {оценка} {оценка} {оценка} ...
...

Выбран пункт 5:

  • Программа запрашивает с консоли имя преподавателя.
  • Если преподаватель с указанным именем найден в списке преподавателей, то запрашивает с консоли имя студента.
  • Если студент с указанным именем найден в списке студентов, то запрашивает с консоли оценку и выдаёт оценку от преподавателя студенту.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Практическое Задание 45.3

Создайте класс "Устройство". Этот класс должен хранить поля:

  • Имя
  • Статус
  • Тип

Для класса определите методы:

  • Конструктор
  • Строковое представление
  • Получение статуса
    • Ничего не принимает.
    • Возвращает статус.
  • Получение типа
    • Ничего не принимает.
    • Возвращает тип.
  • Включить устройство
    • Ничего не принимает.
    • Если устройство выключено, то меняет статус на включено.
  • Выключить устройство
    • Ничего не принимает.
    • Если устройство включено, то меняет статус на выключено.

Создайте класс "Умный свет", который наследуется от класса "Устройство". Этот класс должен хранить поля:

  • Текущий цвет

Для класса определите методы:

  • Конструктор, где тип – 1
  • Строковое представление
  • Установка цвета
    • Принимает строку цвета.
    • Выставляет цвет.

Создайте класс "Умный термостат", который наследуется от класса "Устройство". Этот класс должен хранить поля:

  • Текущая температура

Для класса определите методы:

  • Конструктор, где тип – 2
  • Строковое представление
  • Установка температуры
    • Принимает число.
    • Выставляет температуру.

Создайте класс "Умный замок", который наследуется от класса "Устройство". Этот класс должен хранить поля:

  • Статус замка

Для класса определите методы:

  • Конструктор, где тип – 3
  • Строковое представление
  • Закрыть замок
    • Ничего не принимает.
    • Если замок открыт, то меняет статус замка на закрытый.
  • Открыть замок
    • Ничего не принимает.
    • Если замок закрыт, то меняет статус замка на открытый.

Практическое Задание 45.4

Используя классы из Практического Задания 45.3, реализуйте следующее меню: 

1. Показать все устройства
2. Добавить устройство
3. Удалить устройство
4. Выбрать устройство
5. Завершить программу

Все устройства хранятся в списке устройств. Устройство – это экземпляр класса "Устройство".

Выбран пункт 1:

  • Программа проходит по каждому устройству и выводит в консоль его строковое представление в виде: 
    {имя} {статус} {уникальные_данные_устройства}
{имя} {статус} {уникальные_данные_устройства}
...

Выбран пункт 2:

  • Программа выводит меню в виде: 
    1. Умный свет
2. Умный термостат
3. Умный замок
  • Программа запрашивает с консоли имя.
  • В зависимости от выбранного пункта программа ещё запрашивает цвет или температуру и создаёт соответствующий экземпляр класса.
  • Программа добавляет устройство в список устройств.

Выбран пункт 3:

  • Программа запрашивает с консоли имя.
  • Если устройство с указанным именем есть в списке устройств, то удаляет из устройство из списка.
  • В зависимости от результата, в консоли выводится соответствующее сообщение.

Выбран пункт 4:

  • Программа запрашивает с консоли имя.
  • Если устройство с указанным именем есть в списке устройств, то выводит в консоль его строковое представление и меню в виде: 
    {имя} {статус} {уникальные_данные_устройства}
=========================================
1. Включить (если выключен) / 1. Выключить (если включен)
2. Сменить цвет (умный свет) / 2. Выставить температуру (умный термостат) / 2. Закрыть (умный замок открыт) / 2. Открыть (умный замок закрыт)
  • При выборе пунктов вызываются соответствующие методы устройства.
