Горячие клавиши для работы в PyCharm.

БАЗОВЫЕ ГОРЯЧИЕ КЛАВИШИ:
Зажать Shift + Нажать F10    Запустить код
Зажать Ctrl + Нажать F5    Перезапустить код
Зажать Ctrl + Нажать F2    Прервать выполнение кода

Зажать Ctrl + Нажать S    Сохранить изменения
Зажать Ctrl + Нажать C    Скопировать выделенное
Зажать Ctrl + Нажать V    Вставить скопированное
Зажать Ctrl + Нажать D    Продублировать строку, на которой находится курсор
Зажать Ctrl + Нажать X    Вырезать выделенное. Вырезанное считается скопированным

Зажать Ctrl + Нажать Z    Отменить последнее изменение
Зажать Сtrl + Зажать Shift + Нажать Z    Вернуть отменённое изменение

Зажать Ctrl + Нажать /    Закомментировать/Раскомментировать выделенное или строку

Зажать Ctrl + Нажать Y    Удалить строку, на которой находится курсор

Зажать Shift + Нажать Стрелку Вправо/Влево    Выделить символы

Зажать Ctrl + Нажать Стрелку Вправо/Влево    Двигать курсор по словам
Зажать Ctrl + Нажать Backspace    Удалить слово
Зажать Shift + Зажать Ctrl + Стрелку Вправо/Влево    Выделить слово

Зажать Ctrl + Нажать W    Выделить вширь с учётом синтаксиса
Зажать Ctrl + Зажать Shift + Нажать W    Отменить предыдущее выделение вширь

Выделить код и Нажать Tab    Сместить код правее
Выделить код, Зажать Shift + Нажать Tab    Сместить код левее


ПРОДВИНУТЫЕ ГОРЯЧИЕ КЛАВИШИ:
Поставить курсор на сущность, Зажать Ctrl + Нажать Q    Получение быстрой документации

Зажать Ctrl + Зажать Alt + Нажать Стрелку Вправо/Влево    Перемещение курсора по истории изменений
Зажать Ctrl + Зажать Shift + Нажать Backspace    Переместить курсор к последнему изменению

Зажать Alt + Зажать Shift + Нажать Стрелку Вверх/Вниз    Двигает всю строку вверх/вниз, можно выделить несколько строк и двигать их этим способом
Зажать Ctrl + Зажать Shift + Нажать Стрелку Вверх/Вниз    Двигает блок кода вверх/вниз

Выделить выражение, затем Зажать Ctrl + Зажать Alt + Нажать V    Помещает выделенное выражение в переменную

Зажать Ctrl + Зажать Shift + Нажать Delete    Выносит внутреннюю часть блока, где стоит курсор, во внешнюю часть
Выделить блок кода, затем Зажать Ctrl + Зажать Alt + Нажать T    "Оборачивает" блок кода

Зажать Alt + Нажать 1    Показать/Скрыть панель проекта
Зажать Alt + Нажать 4    Показать/Скрыть панель консоли
Зажать Alt + Нажать 6    Показать/Скрыть панель проблем
Зажать Alt + Нажать 7    Показать/Скрыть панель структур
Зажать Alt + Нажать F12    Показать/Скрыть панель терминала (аналог CMD)

Зажать Shift + Нажать F9    Запустить код в режиме дебага

Во время дебага Зажать Ctrl + Нажать F2    Остановить дебаг
Во время дебага Нажать F8    Перешагнуть блок
Во время дебага Зажать F7    Войти внутрь блока
Во время дебага Зажать Shift + Нажать F8    Выйти из блока
Во время дебага Нажать F9    Продолжить выполнение

Зажать Ctrl + Нажать F    Поиск в текущем файле
Зажать Ctrl + Нажать R    Поиск и замена в текущем файле

Зажать Ctrl + Зажать Shift + Нажать F    Поиск в проекте
Зажать Ctrl + Зажать Shift + Нажать R    Поиск и замена в проекте

Поставить курсор на сущность, Зажать Alt + Нажать F7    Показывает, где используется

Нажать по файлу/папке/проекту, Зажать Shift + Нажать F6    Переименовать файл/папку/проект

Зажать Ctrl + Нажать E    Показать недавние файлы
Зажать Ctrl + Зажать Shift + Нажать E    Показать места, где были недавние изменения

Зажать Alt + Нажать Insert    Генератор кода

Поставить курсор на названии, Зажать Shift + Нажать F6   Быстрая смена всех таких же имён

Два раза нажать Shift    Поиск

Выделить код, Зажать Правый Ctrl + Зажать Правый Alt + Нажать M    Быстро создать функцию
Поставить курсор на названии функции, Зажать Ctrl + Зажать Alt + Нажать N    Удаляет описание функции, но заменяет её вызов на её описание

Поставть курсор на значении в описании функции, Зажать Ctrl + Зажать Alt + Нажать P    Создаёт у функции параметр, заменяет значения на созданный параметр
