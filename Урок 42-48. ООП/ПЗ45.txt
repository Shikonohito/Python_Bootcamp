Практическое Задание 45.1
Создайте класс "Устройство". Этот класс должен хранить поля для названия и времени.
Для класса определите строковое представление, конструктор и свойства.
Для класса определите метод "Использовать", который просто выводит в консоль "Использование".

Создайте ещё 3 класса, которые наследуются от класса "Устройство" - "Кофемашина", "Тостер" и "Посудомойка".
Переопределите для них метод "Использовать" по своему усмотрению.
Переопределите для них конструктор и строковое представление.

Реализуйте следующее меню:
1. Выбрать кофемашину
2. Выбрать тостер
3. Выбрать посудомойка

При выборе любого пункта, программа создаёт объект выбранного класса и вызывает у него метод использования.


Практическое Задание 45.2
Создайте класс "Конвертер".
Для класса определите 2 статических метода конвертации, например, перевода метров в указанную единицу и перевода цельсия в указанную единицу.

Затем реализуйте следующее меню:
1. Метры
2. Цельсий

При выборе единицы измерения программа должна предлагать меню по выбору конвертации, а затем запрашивать количество единиц и выводить результат конвертации.
