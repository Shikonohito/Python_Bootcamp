Практическое Задание 44.1

Создайте класс "Транспорт". Этот класс должен хранить поля для идентификатора, бренда, марки, максимальной скорости и количества пассажиров.
Для класса определите строковое представление, конструктор и свойства.

Создайте класс "Автомобиль". Этот класс должен наследоваться от класса "Транспорт" и хранить поля для цены за км и рейтинг.
Для класса определите строковое представление, конструктор и свойства.
Для класса определите метод получения общей стоимости за указанные км.

Создайте класс "Автобус". Этот класс должен наследоваться от класса "Транспорт" и хранить поля для электронной оплаты (True/False), списка остановок и цены за остановку.
Для класса определите строковое представление, конструктор и свойства.
Для класса определите метод добавления, изменения и удаления остановки.
Для класса определите метод получения общей стоимости за указанные начальную и конечную остановку.

Создайте класс "Самолёт". Этот класс должен наследоваться от класса "Транспорт" и хранить поля для максимального объёма груза (куб. м), максимального веса груза (кг) и цены за кг груза.
Для класса определите строковое представление, конструктор и свойства.
Для класса определите метод получения общей стоимости за указанные объём и вес груза.

Практическое Задание 44.2

Использовав классы из Практического Задания 44.1, создайте по 2 автомобиля и добавьте их в список автомобилей. Проделайте то же самое для автобусов и самолётов.

Реализуйте следующее меню:
1. Показать весь имеющийся транспорт
2. Заказать перевозку пассажиров автомобилем
3. Заказать перевозку груза самолётом
4. Определить стоимость поездки на автобусе
5. Завершить программу

При выборе пункта 1 программа показывает весь транспорт в виде: 

АВТОМОБИЛИ
...
...
=============================================
АВТОБУСЫ
...
...
=============================================
САМОЛЁТЫ
...
...

При выборе пункта 2 программа запрашивает с консоли количество пассажиров и количество км. Затем программа выводит в консоль список автомобилей, которые могут перевозить указанное количество пассажиров, и общую стоимость поездки каждого доступного автомобиля.

При выборе пункта 3 программа запрашивает с консоли количество объёма и веса груза. Затем программа выводит в консоль список самолётов, которые могут перевозить указанное количество объёма и веса груза, а также выводит общую стоимость каждого доступного самолёта.

При выборе пункта 4 программа запрашивает с консоли начальную остановку и конечную остановку. Затем программа выводит в консоль список автобусов, которые содержат указанные остановки, и общую стоимость поездки.
