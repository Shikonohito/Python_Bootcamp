Практическое Задание 3.1

Сделайте скриншот экрана и сохраните его как файл на рабочем столе. Затем создайте текстовый документ на рабочем столе, выделите только что созданный документ и скриншот, затем заархивируйте их.

Практическое Задание 3.2

Посчитайте общую минимальную мощность, которая потребуется для следующей сборки ПК:
— Процессор Intel Core I5 12400f
— Видеокарта RTX 4060 Inno3d compact
— ОЗУ 16gb ddr4 3200 G.Skill Aegis (2 модуля по 8гб)
— Материнская плата Gigabyte b760m ds3h ddr4
— SSD NVME 1tb XPG SX6000 LITE
— Кулер ID-Cooling SE-214 XT Basic
— Корпус PowerCase Mistral Micro D3B ARGB

Укажите, какую мощность Вы бы порекомендовали для данной сборки и почему.

Практическое Задание 3.3

— Выбран CPU
— Выбран CPU Cooler
— Выбран Motherboard
— Выбран Memory
— Выбран Storage
— Выбран Video Card
— Выбран Case
— Выбран Power Supply
