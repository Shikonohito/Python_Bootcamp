Задание 1.
Напишите функцию, которая принимает строку, и возвращает True, если в строке есть цифры, или False в противном случае.

Задание 2.
Напишите функцию, которая принимает строку, и возвращает True, если строка состоит ТОЛЬКО из английских букв, или False в противном случае.

Задание 3.
Напишите функцию, которая принимает строку, переводит все английские заглавные буквы в строчные и возвращает полученную строку.

Задание 4.
Напишите функцию, которая принимает строку и символ, и возвращает индекс первого вхождения символа в строку.
Если указанного символа нет, то функция должна вернуть -1.

Задание 5.
Напишите функцию, которая принимает список, элемент для поиска и индекс, с которого искать.
Функция возвращает индекс первого вхождения элемента в список.
Если указанного элемента нет, то функция должна вернуть -1. Учесть случай, когда получаемый индекс больше размера списка.

Задание 6.
Напишите функцию, которая принимает список и элемент для поиска. Функция возвращает количество найденных элементов в списке.

Задание 7.
Написать рекурсивную функцию нахождения наибольшего общего делителя двух целых чисел.

Задание 8.
Написать рекурсивную функцию нахождения минимального значения из списка чисел.
