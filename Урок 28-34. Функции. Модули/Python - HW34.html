Домашнее Задание 34.1

Реализуйте следующее меню: 

1. Показать список студентов
2. Показать студента по идентификатору
3. Добавить студента в список
4. Удалить студента из списка по идентификатору
5. Изменить студента по идентификатору
6. Найти студентов по Фамилии и Имени
7. Определить среднюю оценку по идентификатору студента
8. Завершить программу

Студент это список следующего вида: 

[{идентификатор}, {фамилия}, {имя}, [ {оценка}, {оценка}, {оценка}, ...]]
По индексам 0, 1, 2 находятся "Идентификатор", "Фамилия", "Имя". По индексу 3 находится список оценок студента.

Список студентов это список следующего вида: 

[[{идентификатор}, {фамилия}, {имя}, [ {оценка}, {оценка}, {оценка}, ...],
 [{идентификатор}, {фамилия}, {имя}, [ {оценка}, {оценка}, {оценка}, ...],
 [{идентификатор}, {фамилия}, {имя}, [ {оценка}, {оценка}, {оценка}, ...],
 ...]

Сперва реализуйте функции, описанные ниже. Везде, где требуется получать данные студента по индексам или изменять данные студента по индексам, вызывайте соответствующую функцию. Функции следующие:

  • get_student_id:
    • Функция принимает студента.
    • Возвращает элемент под индексом 0.
  • get_student_surname:
    • Функция принимает студента.
    • Возвращает элемент под индексом 1.
  • get_student_name:
    • Функция принимает студента.
    • Возвращает элемент под индексом 2.
  • get_student_grades:
    • Функция принимает студента.
    • Возвращает элемент под индексом 3.
  • set_student_id:
    • Функция принимает студента и идентификатор.
    • Заменяет значение под индексом 0 на новое.
    • Ничего не возвращает.
  • set_student_surname:
    • Функция принимает студента и фамилию.
    • Заменяет значение под индексом 1 на новое.
    • Ничего не возвращает.
  • set_student_name:
    • Функция принимает студента и имя.
    • Заменяет значение под индексом 2 на новое.
    • Ничего не возвращает.
  • set_student_grades:
    • Функция принимает студента и список оценок.
    • Заменяет значение под индексом 3 на новое.
    • Ничего не возвращает.

Выбран пункт 1:

  • Программа выводит в консоль список студентов в следующем формате: 
    {идентификатор} {фамилия} {имя} - {оценка} {оценка} {оценка} ...
{идентификатор} {фамилия} {имя} - {оценка} {оценка} {оценка} ...
...
  • Реализуйте через функцию student_list_to_str_list:
    • Функция принимает список студентов.
    • Применяет функцию student_to_str (см. ниже) для каждого студента.
    • Строковое представление студента добавляет в новый список.
    • Возвращает список студентов, который содержит их строковое представление.
  • Реализуйте функцию student_to_str:
    • Функция принимает студента.
    • Возвращает строку в виде "{идентификатор} {фамилия} {имя} - {оценка} {оценка} {оценка} ..."

Выбран пункт 2:

  • Программа запрашивает с консоли идентификатор студента.
  • Если найден студент по идентификатору, то выводит в консоль строковое представление найденного студента.
  • В противном случае выводит в консоль "Студент не найден.".
  • Реализуйте через функцию get_student:
    • Функция принимает список студентов и идентификатор студента.
    • Находит индекс с помощью функции get_student_index (см. ниже).
    • Если по найденному индексу существует студент, то возвращает найденного студента.
    • В противном случае возвращает None.
  • Реализуйте функцию get_student_index:
    • Функция принимает список студентов и идентификатор студента.
    • Сверяет идентификатор с идентификатором каждого студента из списка.
    • Если есть совпадение, то возвращает индекс студента с указанным идентификатором.
    • Если не было ни одного совпадения, то возвращает -1.

Выбран пункт 3:

  • Программа запрашивает с консоли идентификатор, фамилию, имя и отдельно строку оценок через пробел.
  • Строку оценок превращает в список чисел.
  • Формирует из введённых данных студента. Реализуйте через функцию create_student:
    • Функция принимает идентификатор, фамилию, имя и список оценок.
    • Из входных данный функция создаёт список, где по индексу 0 находится идентификатор, по индексу 1 находится фамилия, по индексу 2 находится имя, по индексу 3 находится список оценок.
    • Возвращает полученный список.
  • Сформированного студента добавляет в список студентов.
  • Выводит в консоль оповещение о результате.
  • Реализуйте через функцию add_student:
    • Функция принимает список студентов и студента.
    • Находит индекс с помощью функции get_student_index.
    • Если индекс студента не найден, то добавляет студента в список студентов и возвращает True.
    • В противном случае возвращает False.

Выбран пункт 4:

  • Программа запрашивает с консоли идентификатор.
  • По идентификатору удаляет студента из списка студентов.
  • Выводит в консоль оповещение о результате.
  • Реализуйте через функцию remove_student:
    • Функция принимает список студентов и идентификатор.
    • Находит индекс с помощью функции get_student_index.
    • Если индекс студента найден, то удаляет студента из списка студентов и возвращает True.
    • В противном случае возвращает False.

Выбран пункт 5:

  • Программа запрашивает с консоли идентификатор.
  • Если студент по указанному идентификатору не найден, то выводит в консоль "Студент не найден.".
  • В противном случае делает копию студента.
  • Реализуйте функцию get_copy:
    • Функция принимает студента.
    • Делает копию студента.
    • Возвращает копию студента.
  • Выводит в консоль исходные данные студента, изменённые данные и меню в виде:
