Практическое Задание 31.1

Напишите программу по приготовлению салата "Мимоза" с использованием разных функций и вызове их через цикл.

Рецепт:

  1. Отварить картофель, морковь, яйца.
  2. Почистить картофель, морковь, яйца.
  3. Отдельно натереть на мелкой тёрке картофель, морковь, белок, желток, сыр.
  4. Выложить в салатницу рыбу с маслом.
  5. Размять вилкой по всей поверхности дна посуды.
  6. Верхушку смазать небольшим количеством майонеза.
  7. Сверху разложить натёртые белки.
  8. Верхушку смазать небольшим количеством майонеза.
  9. Сверху разложить натёртую морковь.
  10. Верхушку смазать небольшим количеством майонеза.
  11. Нарезать мелко лук.
  12. Залить кипятком нарезанный лук.
  13. Дать нарезанному луку настояться в течении 5-10 минут.
  14. Сверху разложить нарезанный лук.
  15. Сверху разложить натёртый картофель.
  16. Верхушку смазать небольшим количеством майонеза.
  17. Сверху разложить натёртый сыр.
  18. Верхушку смазать небольшим количеством майонеза.
  19. Сверху разложить натёртый желток.

Функции:

  1. boil(<список_продуктов>, <индексы_продуктов>)
    Возвращает строку "Отварить <название_продукта>, <название_продукта>, ... ." и меняет название продуктов по индексам на "варёный <название_продукта>".
  2. peel(<список_продуктов>, <индексы_продуктов>)
    Возвращает строку "Почистить <название_продукта>, <название_продукта>, ... .".
  3. grater(<вместе/отдельно>, <мелко/средне/крупно>, <список_продуктов>, <индексы_продуктов>)
    Возвращает функцию grate, которая описана внутри него.
  4. grate(<список_продуктов>, <индексы_продуктов>)
    Возвращает строку "<Вместе/Отдельно> натереть на <мелкой/средней/крупной> тёрке <название_продукта>, <название_продукта>, ... ." и меняет название продуктов по индексам на "натёртый <название_продукта>".
  5. cutter(<мелко/средне/крупно>, <список_продуктов>, <индексы_продуктов>)
    Возвращает функцию chop, которая описана внутри него.
  6. chop(<список_продуктов>, <индексы_продуктов>)
    Возвращает строку "<Мелко/Средне/Крупно> нарезать <название_продукта>, <название_продукта>, ... ." и меняет название продуктов по индексам на "нарезанный <название_продукта>".
  7. put(<сверху/сбоку/без_разницы>, <смазать/разложить/без_разницы>, <список_продуктов>, <индексы_продуктов>)
    Возвращает функцию place, которая описана внутри него.
  8. place(<список_продуктов>, <индексы_продуктов>)
    Возвращает строку "<Смазать/Разложить/Положить> <Сверху/Сбоку> <название_продукта>, <название_продукта>, ... .".
  9. cooking_steps(<название_блюда>)
    Возвращает список функций.
  10. cook(<название_блюда>)
    Вызывает cooking_steps и проходится по ним. Затем выводит в консоль "Готово".

Практическое Задание 31.2

Напишите программу, которая по-разному приветствует пользователей. Создайте список users, который будет содержать имена "Teston", "Иван", "Kate". Создайте три функции, которые принимают строку имени и возвращают строки приветствия. Первая функция возвращает "Доброго времени суток, магистр <Имя>.", вторая функция возвращает "Здравствуйте, лорд <Имя>.", третья функция - "Добро пожаловать, <Имя>.".

Напишите ещё одну функцию, которая принимает список имён и список функций приветствий, затем функция в цикле вызывает функции приветствий, передавая им имена.
