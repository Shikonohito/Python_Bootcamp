Практическое Задание 31.1

Реализуйте следующее меню: 

1. Показать список температур
2. Создать новый список
3. Добавить температуру
4. Показать максимальную температуру из списка
5. Показать минимальную температуру из списка
6. Показать среднюю температуру из списка
7. Завершить программу

Выбран пункт 1:

  • Программа выводит в консоль данные списка температур в одной строке.
  • Реализуйте через функцию show_list:
    • Функция принимает список температур.
    • Выводит в консоль в одну строку через пробел температуры.
    • Ничего не возвращает.

Выбран пункт 2:

  • Программа запрашивает с консоли в одну строчку температуры через пробел.
  • Введённые температуры приводит к числам, помещает в список и заменяет список температур.
  • Реализуйте через функцию input_temperature_list:
    • Функция ничего не принимает.
    • Запрашивает с консоли в одну строчку температуры через пробел.
    • Приводит данные к числам и помещает в новый список.
    • Возвращает новый список.

Выбран пункт 3:

  • Программа запрашивает с консоли температуру.
  • Введённую температуру помещает в список температур.
  • Реализуйте через функцию input_temperature:
    • Функция принимает список температур.
    • Запрашивает с консоли температуру.
    • Добавляет температуру в список температур.
    • Ничего не возвращает.

Выбран пункт 4:

  • Программа выводит в консоль максимальную температуру.
  • Реализуйте через функцию get_max_temperature:
    • Функция принимает список температур.
    • Если список пустой, то возвращает None.
    • В противном случае находит максимальную температуру и возвращает её.

Выбран пункт 5:

  • Программа выводит в консоль минимальную температуру.
  • Реализуйте через функцию get_min_temperature:
    • Функция принимает список температур.
    • Если список пустой, то возвращает None.
    • В противном случае находит минимальную температуру и возвращает её.

Выбран пункт 6:

  • Программа выводит в консоль среднюю температуру.
  • Реализуйте через функцию get_avg_temperature:
    • Функция принимает список температур.
    • Если список пустой, то возвращает None.
    • В противном случае находит среднюю температуру и возвращает её.

Практическое Задание 31.2

Реализуйте следующее меню: 

1. Показать список оценок
2. Создать новый список
3. Добавить оценку
4. Удалить оценку по порядковому номеру
5. Изменить оценку по порядковому номеру
6. Показать среднюю оценку из списка
7. Завершить программу

Выбран пункт 1:

  • Программа выводит в консоль данные списка оценок в одной строке.
  • Реализуйте через функцию show_list:
    • Функция принимает список оценок.
    • Выводит в консоль в одну строку через пробел оценки.
    • Ничего не возвращает.

Выбран пункт 2:

  • Программа запрашивает с консоли в одну строчку оценки через пробел.
  • Введённые оценки приводит к числам, помещает в список и заменяет список оценок.
  • Реализуйте через функцию input_grade_list:
    • Функция ничего не принимает.
    • Запрашивает с консоли в одну строчку оценки через пробел.
    • Приводит данные к числам
    • Если проверка оценки с помощью функции is_correct_grade (см. ниже) дала True, то помещает оценку в новый список.
    • Возвращает новый список.
  • Реализуйте функцию is_correct_grade:
    • Функция принимает оценку.
    • Если оценка в промежутке от 1 до 12 включительно, то возвращает True.
    • В противном случае возвращает False.

Выбран пункт 3:

  • Программа запрашивает с консоли оценку.
  • Если проверка оценки с помощью функции is_correct_grade дала True, то помещает оценку в список оценок.
  • В зависимости от результата, выводит в консоль сообщение.
  • Реализуйте через функцию add_grade:
    • Функция принимает список оценок и оценку.
    • Добавляет оценку в список оценок.
    • Ничего не возвращает.

Выбран пункт 4:

  • Программа запрашивает с консоли порядковый номер оценки (не индекса!).
  • Если проверка индекса с помощью функции is_correct_index (см. ниже) дала True, то удаляет оценку по индексу.
  • Реализуйте через функцию remove_grade:
    • Функция принимает список оценок и индекс.
    • Удаляет оценку по индексу.
    • Ничего не возвращает.
  • Реализуйте функцию is_correct_index:
    • Функция принимает список оценок и индекс.
    • Если индекс в промежутке от 0 до размера списка оценок, то возвращает True.
    • В противном случае возвращает False.

Выбран пункт 5:

  • Программа запрашивает с консоли порядковый номер оценки (не индекса!).
  • Если проверка индекса с помощью функции is_correct_index дала True, то запрашивает с консоли оценку.
  • Если проверка оценки с помощью функции is_correct_grade дала True, то заменяет оценку по порядковому номеру.
  • Реализуйте через функцию change_grade:
    • Функция принимает список оценок, индекс и оценку.
    • Заменяет оценку по индексу.
    • Ничего не возвращает.

Выбран пункт 6:

  • Программа выводит в консоль среднюю арифметическую оценку списка.
  • Реализуйте через функцию get_avg_grade:
    • Функция принимает список оценок.
    • Если список пустой, то возвращает None.
    • В противном случае возвращает среднюю арифметическую оценку списка.

Практическое Задание 31.3

Напишите программу, которая по-разному приветствует пользователей. Создайте список users, который будет содержать имена "Teston", "Иван", "Kate". Создайте три функции, которые принимают строку имени и возвращают строки приветствия. Первая функция возвращает "Доброго времени суток, магистр {имя}.", вторая функция возвращает "Здравствуйте, лорд {имя}.", третья функция – "Добро пожаловать, {имя}.".

Создайте ещё одну функцию, которая принимает список имён и список функций приветствий, затем функция в цикле вызывает функции приветствий, передавая им имена.

Практическое Задание 31.4

Напишите программу по приготовлению салата "Мимоза" с использованием разных функций и вызове их через цикл.

Рецепт:

  • Отварить картофель, морковь, яйца.
  • Почистить картофель, морковь, яйца.
  • Отдельно натереть на мелкой тёрке картофель, морковь, белок, желток, сыр.
  • Выложить в салатницу рыбу с маслом.
  • Размять вилкой по всей поверхности дна посуды.
  • Верхушку смазать небольшим количеством майонеза.
  • Сверху разложить натёртые белки.
  • Верхушку смазать небольшим количеством майонеза.
  • Сверху разложить натёртую морковь.
  • Верхушку смазать небольшим количеством майонеза.
  • Нарезать мелко лук.
  • Залить кипятком нарезанный лук.
  • Дать нарезанному луку настояться в течении 5-10 минут.
  • Сверху разложить нарезанный лук.
  • Сверху разложить натёртый картофель.
  • Верхушку смазать небольшим количеством майонеза.
  • Сверху разложить натёртый сыр.
  • Верхушку смазать небольшим количеством майонеза.
  • Сверху разложить натёртый желток.

Функции:

  • boil({список_продуктов}, {индексы_продуктов}):
    • Возвращает строку "Отварить {название_продукта}, {название_продукта}, ... .".
    • Меняет название продуктов по индексам на "варёный {название_продукта}".
  • peel({список_продуктов}, {индексы_продуктов})
    • Возвращает строку "Почистить {название_продукта}, {название_продукта}, ... .".
  • grater({слово_вместе/отдельно}, {слово_мелко/средне/крупно}, {список_продуктов}, {индексы_продуктов})
    • Возвращает функцию grate, которая описана внутри него.
    • grate({список_продуктов}, {индексы_продуктов})
      • Возвращает строку "{Вместе/Отдельно} натереть на {мелкой/средней/крупной} тёрке {название_продукта}, {название_продукта}, ... .".
      • Меняет название продуктов по индексам на "натёртый {название_продукта}".
  • cutter({слово_мелко/средне/крупно}, {список_продуктов}, {индексы_продуктов})
    • Возвращает функцию chop, которая описана внутри него.
    • chop({список_продуктов}, {индексы_продуктов})
      • Возвращает строку "{Мелко/Средне/Крупно} нарезать {название_продукта}, {название_продукта}, ... .".
      • Меняет название продуктов по индексам на "нарезанный {название_продукта}".
  • put ({слово_сверху/сбоку/без_разницы}, {слово_смазать/разложить/без_разницы}, {список_продуктов}, {индексы_продуктов})
    • Возвращает функцию place, которая описана внутри него.
    • place({список_продуктов}, {индексы_продуктов})
      • Возвращает строку "{Смазать/Разложить/Положить} {Сверху/Сбоку} {название_продукта}, {название_продукта}, ... .".
  • get_cooking_funcs({название_блюда})
    • Возвращает список требуемых функций.
  • cook({название_блюда})
    • Вызывает cooking_steps.
    • Проходится по полученным функциям и вызывает их.
    • Выводит в консоль результаты.
