Практическое Задание 31.1

Реализуйте следующее меню: 

1. Показать список температур
2. Создать новый список
3. Добавить температуру
4. Показать максимальную температуру из списка
5. Показать минимальную температуру из списка
6. Показать среднюю температуру из списка
7. Завершить программу

При выборе пункта 1 программа должна выводить в консоль все температуры из списка в одну строчку.
Реализуйте через функцию show_list:
— Функция принимает список температур.
— Выводит в консоль в одну строку через пробел температуры.
— Ничего не возвращает.

При выборе пункта 2 программа должна запросить с консоли в одну строчку температуры через пробел и поместить в список.
Реализуйте через функцию input_temperature_list:
— Функция ничего не принимает.
— Запрашивает с консоли в одну строчку температуры через пробел.
— Конвертирует данные в числа и помещает в список.
— Возвращает полученный список.

При выборе пункта 3 программа должна запросить с консоли температуру и поместить в список.
Реализуйте через функцию input_temperature:
— Функция принимает список температур.
— Запрашивает с консоли температуру.
— Добавляет температуру в список.
— Ничего не возвращает.

При выборе пункта 4 программа должна показать в консоли максимальную температуру из списка.
Реализуйте через функцию get_max_temperature:
— Функция принимает список температур.
— Если список пустой, то возвращает None. В противном случае находит максимальную температуру и возвращает её.

При выборе пункта 5 программа должна показать в консоли минимальную температуру из списка.
Реализуйте через функцию get_min_temperature:
— Функция принимает список температур.
— Если список пустой, то возвращает None. В противном случае находит минимальную температуру и возвращает её.

При выборе пункта 6 программа должна показать в консоли среднюю температуру из списка.
Реализуйте через функцию get_avg_temperature:
— Функция принимает список температур.
— Если список пустой, то возвращает None. В противном случае находит среднюю температуру и возвращает её.

Практическое Задание 31.2

Реализуйте следующее меню: 

1. Показать список оценок
2. Создать новый список
3. Добавить оценку
4. Удалить оценку по порядковому номеру
5. Изменить оценку по порядковому номеру
6. Показать среднюю оценку из списка
7. Завершить программу

При выборе пункта 1 программа должна выводить в консоль все оценки из списка в одну строчку.
Реализуйте через функцию show_list:
— Функция принимает список оценок.
— Выводит в консоль в одну строку через пробел оценки.
— Ничего не возвращает.

При выборе пункта 2 программа должна запросить с консоли в одну строчку оценки через пробел и поместить в список.
Реализуйте через функцию input_grade_list:
— Функция ничего не принимает.
— Запрашивает с консоли в одну строчку температуры через пробел.
— Конвертирует данные в числа.
— Проверяет каждую оценку с помощью функции is_correct_grade (см. ниже). Если оценка проходит проверку, то помещает её в список.
— Возвращает полученный список.

Функция is_correct_grade:
— Принимает оценку.
— Если оценка в промежутке от 1 до 12 включительно, то возвращает True. В противном случае возвращает False.

При выборе пункта 3 программа должна запросить с консоли оценку и поместить в список.
Реализуйте через функцию add_grade:
— Функция принимает список оценок и оценку.
— Проверяет оценку с помощью функции is_correct_grade. Если оценка проходит проверку, то помещает её в список и возвращает True. В противном случае возвращает False.

При выборе пункта 4 программа должна запросить с консоли порядковый номер оценки и удалить оценку.
Реализуйте через функцию remove_grade:
— Функция принимает список оценок и индекс.
— Проверяет индекс с помощью функции is_correct_index (см. ниже). Если индекс проходит проверку, то удаляет оценку по индексу из списка и возвращает True. В противном случае возвращает False.

Функция is_correct_index:
— Принимает список оценок и индекс.
— Если индекс в промежутке от 0 до {размера_списка}, то возвращает True. В противном случае возвращает False.

При выборе пункта 5 программа должна запросить с консоли порядковый номер оценки и оценку. Затем программа должна изменить оценку по порядковому номеру на другую оценку.
Реализуйте через функцию change_grade:
— Функция принимает список оценок, индекс и новую оценку.
— Проверяет оценку с помощью is_correct_grade и индекс с помощью is_correct_index. Если всё в порядке, то изменяет оценку по индексу на новую оценку и возвращает True. В противном случае возвращает False.

При выборе пункта 6 программа должна показать в консоли среднюю оценку из списка.
Реализуйте через функцию get_avg_grade:
— Функция принимает список оценок.
— Если список пустой, то возвращает None. В противном случае находит среднюю оценку и возвращает её.

Практическое Задание 31.3

Напишите программу, которая по-разному приветствует пользователей. Создайте список users, который будет содержать имена "Teston", "Иван", "Kate". Создайте три функции, которые принимают строку имени и возвращают строки приветствия. Первая функция возвращает "Доброго времени суток, магистр <Имя>.", вторая функция возвращает "Здравствуйте, лорд <Имя>.", третья функция - "Добро пожаловать, <Имя>.".

Напишите ещё одну функцию, которая принимает список имён и список функций приветствий, затем функция в цикле вызывает функции приветствий, передавая им имена.

Практическое Задание 31.4

Напишите программу по приготовлению салата "Мимоза" с использованием разных функций и вызове их через цикл.

Рецепт:

  1. Отварить картофель, морковь, яйца.
  2. Почистить картофель, морковь, яйца.
  3. Отдельно натереть на мелкой тёрке картофель, морковь, белок, желток, сыр.
  4. Выложить в салатницу рыбу с маслом.
  5. Размять вилкой по всей поверхности дна посуды.
  6. Верхушку смазать небольшим количеством майонеза.
  7. Сверху разложить натёртые белки.
  8. Верхушку смазать небольшим количеством майонеза.
  9. Сверху разложить натёртую морковь.
  10. Верхушку смазать небольшим количеством майонеза.
  11. Нарезать мелко лук.
  12. Залить кипятком нарезанный лук.
  13. Дать нарезанному луку настояться в течении 5-10 минут.
  14. Сверху разложить нарезанный лук.
  15. Сверху разложить натёртый картофель.
  16. Верхушку смазать небольшим количеством майонеза.
  17. Сверху разложить натёртый сыр.
  18. Верхушку смазать небольшим количеством майонеза.
  19. Сверху разложить натёртый желток.

Функции:

  1. boil({список_продуктов}, {индексы_продуктов})
    Возвращает строку "Отварить {название_продукта}, {название_продукта}, ... ." и меняет название продуктов по индексам на "варёный {название_продукта}".
  2. peel({список_продуктов}, {индексы_продуктов})
    Возвращает строку "Почистить {название_продукта}, {название_продукта}, ... .".
  3. grater({вместе/отдельно}, {мелко/средне/крупно}, {список_продуктов}, {индексы_продуктов})
    Возвращает функцию grate, которая описана внутри него.
  4. grate({список_продуктов}, {индексы_продуктов})
    Возвращает строку "{Вместе/Отдельно} натереть на {мелкой/средней/крупной} тёрке {название_продукта}, {название_продукта}, ... ." и меняет название продуктов по индексам на "натёртый {название_продукта}".
  5. cutter({мелко/средне/крупно}, {список_продуктов}, {индексы_продуктов})
    Возвращает функцию chop, которая описана внутри него.
  6. chop({список_продуктов}, {индексы_продуктов})
    Возвращает строку "{Мелко/Средне/Крупно} нарезать {название_продукта}, {название_продукта}, ... ." и меняет название продуктов по индексам на "нарезанный {название_продукта}".
  7. put({сверху/сбоку/без_разницы}, {смазать/разложить/без_разницы}, {список_продуктов}, {индексы_продуктов})
    Возвращает функцию place, которая описана внутри него.
  8. place({список_продуктов}, {индексы_продуктов})
    Возвращает строку "{Смазать/Разложить/Положить} {Сверху/Сбоку} {название_продукта}, {название_продукта}, ... .".
  9. cooking_steps({название_блюда})
    Возвращает список функций.
  10. cook({название_блюда})
    Вызывает cooking_steps и проходится по ним. Затем выводит в консоль "Готово".
