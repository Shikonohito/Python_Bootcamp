Задание 31.1
Напишите программу по приготовлению салата "Мимоза" с использованием разных функций и вызове их через цикл.

Салат "Мимоза"
Ингредиенты
Рыбные консервы в масле - 200 Грамм
Картофель - 300 Грамм (4 средних картофелины)
Морковь - 200 Грамм (4 средних морковки)
Лук - 100 Грамм (2 средних луковицы)
Сыр - 150 Грамм (Твёрдого сорта)
Майонез - 250 Грамм
Яйца - 4 Штуки

Рецепт
Отварить картофель, морковь, яйца.
Почистить картофель, морковь, яйца.
Отдельно натереть на мелкой тёрке картофель, морковь, белок, желток, сыр.
Выложить в салатницу рыбу с маслом.
Размять вилкой по всей поверхности дна посуды.
Верхушку смазать небольшим количеством майонеза.
Сверху разложить натёртые белки.
Верхушку смазать небольшим количеством майонеза.
Сверху разложить натёртую морковь.
Верхушку смазать небольшим количеством майонеза.
Нарезать мелко лук.
Залить кипятком нарезанный лук.
Дать нарезанному луку настояться в течении 5-10 минут.
Сверху разложить нарезанный лук.
Сверху разложить натёртый картофель.
Верхушку смазать небольшим количеством майонеза.
Сверху разложить натёртый сыр.
Верхушку смазать небольшим количеством майонеза.
Сверху разложить натёртый желток.


boil(список_продуктов, индексы_продуктов) - возвращает строку "Отварить <название_продукта>, <название_продукта>, ... ." и меняет название продуктов по индексам на "варёный <название_продукта>".
peel(список_продуктов, индексы_продуктов) - возвращает строку "Почистить <название_продукта>, <название_продукта>, ... ."
grater(вместе/отдельно, мелко/средне/крупно, список_продуктов, индексы_продуктов) - возвращает функцию grate, которая описана внутри него.
grate(список_продуктов, индексы_продуктов) возвращает строку "<Вместе/Отдельно> натереть на <мелкой/средней/крупной> тёрке <название_продукта>, <название_продукта>, ... ." и меняет название продуктов по индексам на "натёртый <название_продукта>".
cutter(мелко/средне/крупно, список_продуктов, индексы_продуктов) - возвращает функцию chop, которая описана внутри него.
chop(список_продуктов, индексы_продуктов) - возвращает строку "<Мелко/Средне/Крупно> нарезать <название_продукта>, <название_продукта>, ... ." и меняет название продуктов по индексам на "нарезанный <название_продукта>".
put(сверху/сбоку/без_разницы, смазать/разложить/без_разницы, список_продуктов, индексы_продуктов) - возвращает функцию place, которая описана внутри него.
place(список_продуктов, индексы_продуктов) - возвращает строку "<Смазать/Разложить/Положить> <Сверху/Сбоку> <название_продукта>, <название_продукта>, ... ."

cooking_steps(название_блюда) - возвращает список функций.

cook(название_блюда)
вызывает cooking_steps и проходится по ним. Затем выводит в консоль "Готово".


Задание 31.2
Напишите программу, которая по-разному приветствует пользователей. Создайте список users, который будет содержать имена "Teston", "Иван", "Kate". Создайте три функции, которые принимают строку имени и возвращают строки приветствия. Первая функция возвращает "Доброго времени суток, магистр <Имя>", вторая функция возвращает "Здравствуйте, лорд <Имя>", третья функция "Добро пожаловать, <Имя>".
Напишите ещё одну функцию, которая принимает список имён и список функций приветствий, затем функция в цикле вызывает функции приветствий, передавая им имена.
