Практическое Задание 30.1

Напишите программу, которая ведёт учёт расходов и доходов. Реализуйте следующее меню: 

1. Показать историю расходов и доходов
2. Показать текущий баланс
3. Добавить доход
4. Добавить расход
5. Завершить программу

В программе должны быть глобальный список и баланс.

Выбран пункт 1:

  • Программа выводит в консоль данные глобального списка на каждой строке.

Выбран пункт 2:

  • Программа выводит в консоль текущий баланс.

Выбран пункт 3:

  • Программа запрашивает с консоли число.
  • Если число положительное, то вызывает функцию add_income:
    • Функция принимает сумму дохода.
    • Добавляет доход в баланс.
    • Добавляет строку "Доход: {доход}" в глобальный список.
    • Ничего не возвращает.
  • В зависимости от результата, в консоль выводится сообщение.

Выбран пункт 4:

  • Программа запрашивает с консоли число.
  • Если число положительное, то вызывает функцию add_expense:
    • Функция принимает сумму расхода.
    • Отнимает расход из баланса.
    • Добавляет строку "Расход: {расход}" в глобальный список.
    • Ничего не возвращает.
  • В зависимости от результата, в консоль выводится сообщение.

Практическое Задание 30.2

Переделайте предыдущую программу так, чтобы функции не использовали глобальные переменные.

Практическое Задание 30.3

Переделайте программу из Практического Задания 30.1 так, чтобы использовалось замыкание, вместо глобальных переменных. Также напишите две функции, которые позволят прочитать данные из списка и баланса, и две функции, которые позволят установить новые данные.
